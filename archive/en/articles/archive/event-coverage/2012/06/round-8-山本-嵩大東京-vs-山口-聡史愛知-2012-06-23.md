Round 8: 山本 嵩大(東京) vs. 山口 聡史(愛知)
================================



 Posted in **Event Coverage**
 on June 23, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_authorpic_larger.jpg)
By Wizards of the Coast












「この会場に、無敗のエルフがいる。」


　海外カバレージ・スタッフの指名により、山本 嵩大がフィーチャー・マッチ・エリアに呼ばれた。


 




|  |
| --- |
|  |
| *山本 嵩大* |

　普段からオリジナルのデッキを使うことにこだわる彼は、今回も脳内とショップでの大会で地道に調整を続けてきた。


**山本** 「自分と同じデッキを使ってくれる人がいないんで、使いこむしかないんですよ。」


　今大会での7連勝は、その努力が実った形だ。


　対するのは山口 聡史。これまでもバブルマッチでたびたびフィーチャーされてきた愛知のプレイヤーである。  

　MOや友人たちとの調整を繰り返し、山本と同じく、無敗で8回戦目を迎えた。  

　デッキはジャンドと親和を仮想敵とし、独自の調整を加えたジャンドである。


　果たして唯一無敗のエルフは、ジャンドを前に全勝を守れるのか。


### ゲーム１


![R8_yamaguchi_VS_yamamoto.jpg](https://media.magic.wizards.com/image_legacy_migration/images/magic/daily/eventsjp/gpyok12/R8_yamaguchi_VS_yamamoto.jpg)


　先手の山本は《[新緑の地下墓地](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%96%B0%E7%B7%91%E3%81%AE%E5%9C%B0%E4%B8%8B%E5%A2%93%E5%9C%B0)》から《寺院の庭/[Temple Garden](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Temple+Garden+) 》をサーチし《[東屋のエルフ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%9D%B1%E5%B1%8B%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》へと繋げる。エルフの定番スタートである。  

　さらに続くターンに《[イラクサの歩哨](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A4%E3%83%A9%E3%82%AF%E3%82%B5%E3%81%AE%E6%AD%A9%E5%93%A8)》を送り込んで、クリーチャー的には盤石の体制。


　しかし、2枚目の土地が置けない。  

　そこで、山口は《[稲妻](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%A8%B2%E5%A6%BB)》を《[東屋のエルフ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%9D%B1%E5%B1%8B%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》に。山本のマナを縛りにかかる。  

　このままではジリ貧となる山本は、2枚目の《[東屋のエルフ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%9D%B1%E5%B1%8B%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》を戦場へ。さらに《[遺産のドルイド](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%81%BA%E7%94%A3%E3%81%AE%E3%83%89%E3%83%AB%E3%82%A4%E3%83%89)》から《[エルフの大ドルイド](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A8%E3%83%AB%E3%83%95%E3%81%AE%E5%A4%A7%E3%83%89%E3%83%AB%E3%82%A4%E3%83%89)》と、エルフらしい展開力を見せつける。


　山口は《闇の腹心/[Dark Confidant](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Dark+Confidant+) 》で《[ヴェールのリリアナ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%B4%E3%82%A7%E3%83%BC%E3%83%AB%E3%81%AE%E3%83%AA%E3%83%AA%E3%82%A2%E3%83%8A)》を手に入れ、マナを立てたままターンを返す。


　ここで山本はしばし考え込み、《[エルフの大ドルイド](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A8%E3%83%AB%E3%83%95%E3%81%AE%E5%A4%A7%E3%83%89%E3%83%AB%E3%82%A4%E3%83%89)》からひねり出したマナを用いて、《[ジョラーガの戦呼び](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A7%E3%83%A9%E3%83%BC%E3%82%AC%E3%81%AE%E6%88%A6%E5%91%BC%E3%81%B3)》（キッカー2）を戦場へ。  

　山口は待ってましたとばかりに、《[ジャンドの魔除け](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A3%E3%83%B3%E3%83%89%E3%81%AE%E9%AD%94%E9%99%A4%E3%81%91)》を合わせ、山本の戦線を完全に破壊。

　さらに返すターン、満を持しての《[血編み髪のエルフ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A1%80%E7%B7%A8%E3%81%BF%E9%AB%AA%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》！  

　見事、続唱が《[稲妻](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%A8%B2%E5%A6%BB)》を呼び込み、これが《[ジョラーガの戦呼び](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A7%E3%83%A9%E3%83%BC%E3%82%AC%E3%81%AE%E6%88%A6%E5%91%BC%E3%81%B3)》を片付けると、山本の場はすっからかんに。


　さらに山口が《[思考囲い](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%80%9D%E8%80%83%E5%9B%B2%E3%81%84)》を撃ち込むと、山本は自らの手札が露わとなることを嫌って、投了。


*山本 0-1 山口*


### ゲーム２


 




|  |
| --- |
|  |
| *山口 聡史* |

　山本はゲーム１と同じくフェッチランドからの《[ラノワールのエルフ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%A9%E3%83%8E%E3%83%AF%E3%83%BC%E3%83%AB%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》と、定番の開幕。さらに《[森のレインジャー](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE%E3%81%AE%E3%83%AC%E3%82%A4%E3%83%B3%E3%82%B8%E3%83%A3%E3%83%BC)》から《[平地](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B9%B3%E5%9C%B0)》を手に入れ、今度はマナトラブルを回避してみせる。  

　そして《[ジャンドの魔除け](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A3%E3%83%B3%E3%83%89%E3%81%AE%E9%AD%94%E9%99%A4%E3%81%91)》を警戒してか、マナを立てたままターンエンド。


　山口、山本が隠している「何か」を暴くために《[思考囲い](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%80%9D%E8%80%83%E5%9B%B2%E3%81%84)》。  

　公開された


* 《[ジョラーガの戦呼び](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A7%E3%83%A9%E3%83%BC%E3%82%AC%E3%81%AE%E6%88%A6%E5%91%BC%E3%81%B3)》
* 《[エルフの大ドルイド](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A8%E3%83%AB%E3%83%95%E3%81%AE%E5%A4%A7%E3%83%89%E3%83%AB%E3%82%A4%E3%83%89)》
* 《[イーオスのレインジャー](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A4%E3%83%BC%E3%82%AA%E3%82%B9%E3%81%AE%E3%83%AC%E3%82%A4%E3%83%B3%E3%82%B8%E3%83%A3%E3%83%BC)》
* 《[殴打頭蓋](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%AE%B4%E6%89%93%E9%A0%AD%E8%93%8B)》
* 《[流刑への道](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%B5%81%E5%88%91%E3%81%B8%E3%81%AE%E9%81%93)》

　から、《[イーオスのレインジャー](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A4%E3%83%BC%E3%82%AA%E3%82%B9%E3%81%AE%E3%83%AC%E3%82%A4%E3%83%B3%E3%82%B8%E3%83%A3%E3%83%BC)》を抜き取る。


　山本、山口のターンエンドに自らの《[森のレインジャー](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE%E3%81%AE%E3%83%AC%E3%82%A4%E3%83%B3%E3%82%B8%E3%83%A3%E3%83%BC)》を《[流刑への道](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%B5%81%E5%88%91%E3%81%B8%E3%81%AE%E9%81%93)》。《[森](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE)》を加えて4マナに到達しようとするが、山口は《[稲妻](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%A8%B2%E5%A6%BB)》を《[ラノワールのエルフ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%A9%E3%83%8E%E3%83%AF%E3%83%BC%E3%83%AB%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》に撃ち込んで、これを許さない。


　しかたなく、山本は《[エルフの大ドルイド](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A8%E3%83%AB%E3%83%95%E3%81%AE%E5%A4%A7%E3%83%89%E3%83%AB%E3%82%A4%E3%83%89)》をプレイ。  

　これも山口は《[稲妻](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%A8%B2%E5%A6%BB)》で処理し、残ったマナで《タルモゴイフ/[Tarmogoyf](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Tarmogoyf+) 》。  

　モダン環境を象徴する緑の怪物は、早くも4/5に育っている。


　山本、戦線を維持すべく《[背教の主導者、エズーリ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%83%8C%E6%95%99%E3%81%AE%E4%B8%BB%E5%B0%8E%E8%80%85%E3%80%81%E3%82%A8%E3%82%BA%E3%83%BC%E3%83%AA)》《[ジョラーガの戦呼び](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A7%E3%83%A9%E3%83%BC%E3%82%AC%E3%81%AE%E6%88%A6%E5%91%BC%E3%81%B3)》（キッカー1）を続けて召喚。


　しかし、次なるターン。山口がプレイしたのは《[オリヴィア・ヴォルダーレン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AA%E3%83%AA%E3%83%B4%E3%82%A3%E3%82%A2%E3%83%BB%E3%83%B4%E3%82%A9%E3%83%AB%E3%83%80%E3%83%BC%E3%83%AC%E3%83%B3)》！  

　これには山本、小さな悲鳴を上げる。




![ã‚ªãƒªãƒ´ã‚£ã‚¢ãƒ»ãƒ´ã‚©ãƒ«ãƒ€ãƒ¼ãƒ¬ãƒ³](http://gatherer.wizards.com/Handlers/Image.ashx?type=card&name=%E3%82%AA%E3%83%AA%E3%83%B4%E3%82%A3%E3%82%A2%E3%83%BB%E3%83%B4%E3%82%A9%E3%83%AB%E3%83%80%E3%83%BC%E3%83%AC%E3%83%B3)

　さらに山口は《[コジレックの審問](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B3%E3%82%B8%E3%83%AC%E3%83%83%E3%82%AF%E3%81%AE%E5%AF%A9%E5%95%8F)》で、山本から後続として用意されていた《[ジョラーガの戦呼び](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A7%E3%83%A9%E3%83%BC%E3%82%AC%E3%81%AE%E6%88%A6%E5%91%BC%E3%81%B3)》を奪う。


　手札をズタズタにされた山本は、《[オリヴィア・ヴォルダーレン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AA%E3%83%AA%E3%83%B4%E3%82%A3%E3%82%A2%E3%83%BB%E3%83%B4%E3%82%A9%E3%83%AB%E3%83%80%E3%83%BC%E3%83%AC%E3%83%B3)》による蹂躙を見守る他なかった。


*山本 2-0 山口*
