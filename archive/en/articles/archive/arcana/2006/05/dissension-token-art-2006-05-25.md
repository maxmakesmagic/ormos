*Dissension* Token Art
======================



 Posted in **Arcana**
 on May 25, 2006 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_authorpic_larger.jpg)
By Wizards of the Coast











The *Dissension*  release for ***Magic**  Online*  is right around the corner! The set features four new pieces of art commissioned for the latest crop of token-generating cards. So, while you drool at the prospect of picking up your [Momir Vig](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Momir+Vig) and Lyzolda avatars (see [Bennie Smith's article today](/en/articles/archive/mtgo-iii-news-and-more-2006-05-25) for more on those), here's an exclusive look at the new token menaces you'll be summoning up shortly. 

![Snake token art](https://media.magic.wizards.com/image_legacy_migration/magic/images/mtgcom/arcana1000/1086_Snake.jpg)  
*Dissension Snake token art (for [Patagia Viper](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Patagia+Viper)) by Christopher Moeller*

![Drake token art](https://media.magic.wizards.com/image_legacy_migration/magic/images/mtgcom/arcana1000/1086_Drake.jpg)  
*Drake token art (for [Leafdrake Roost](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Leafdrake+Roost)) by Nick Percival*

![Wurm token art](https://media.magic.wizards.com/image_legacy_migration/magic/images/mtgcom/arcana1000/1086_Elemental.jpg)  
*Elemental token art (from [research // development](http://gatherer.wizards.com/Pages/Card/Details.aspx?&name=research%2B%252F%252F%2Bdevelopment), of [research // development](http://gatherer.wizards.com/Pages/Card/Details.aspx?&name=research%2B%252F%252F%2Bdevelopment)) by Greg Staples*

![Wurm token art](https://media.magic.wizards.com/image_legacy_migration/magic/images/mtgcom/arcana1000/1086_Bird.jpg)  
*Bird token art (for [Pride of the Clouds](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Pride+of+the+Clouds) and [Dovescape](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Dovescape)) by Shishizaru*

*Dissension* goes on sale May 30th for ***Magic** Online*, and release tournaments start up June 1st, so you'll be swinging in with your own copies of these creatures in no time. See you then!
