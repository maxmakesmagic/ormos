グランプリ横浜、押し寄せるプレイヤーと会場風景
=======================



 Posted in **Event Coverage**
 on June 22, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_authorpic_larger.jpg)
By Wizards of the Coast











　グランプリ横浜の会場、パシフィコ横浜では本日開催のイベントが多いそうで、たくさんの人の波がここに向かって流れ込んでいきます。開会を間近に控えた会場の模様をお届けしてまいりましょう。 


　まず先に、会場への道のりをご紹介しておきます。  
 　アクセスはいくつかありますが、一番「みなとみらい」駅からの道案内となります。 







|  |
| --- |
|  |
| *みなとみらい駅で下車。改札を出て左手へ進みましょう。* |






|  |
| --- |
|  |
| *長ーいエスカレーターを上ります。右の巨大な壁に詩が書いてあって、いつ見てもカッコイイです。* |






|  |
| --- |
|  |
| *エスカレーターを一番上まで上りきったら、道なりに左へ進みます。ちなみにこの右側にフードコートがあります。* |






|  |
| --- |
|  |
| *外に出ます。正面にホテルがあり、人の流れに沿って左に曲がればすぐに「展示ホール」の表示が見えてきます。* |






|  |
| --- |
|  |
| *手近な入口から中に入りましょう。目指すは「ホールA」です。* |






|  |
| --- |
|  |
| *看板にしたがって階段を降りれば、会場はすぐそこです。* |






|  |
| --- |
|  |
| *隣のホールBにすごい行列ができているかもしれませんが、こちらはファイナルファンタジーXIのイベントなので、お間違えのないように。* |






|  |
| --- |
|  |
| *会場を出てすぐ隣にコンビニがあります。ただ、この人出により品物がないタイミングもありえますのでご注意ください。* |






|  |
| --- |
|  |
| *それでは会場入り口を入ってみましょう。当日受付の行列がホールの外にまで延びていました。* |






|  |
| --- |
|  |
| *会場をとりまくようにショップブースが並んでおり、すでにすごい人だかりです。のちほどショップ紹介記事も掲載予定です。* |






|  |
| --- |
|  |
| *会場右手のWPNブースではDCIメンバー登録などが行なえ、今話題の「デュエルズ・オブ・プレインズウォーカー2013」の試遊もできます。* |






|  |
| --- |
|  |
| *アーティストサイン会のブースもただいま準備中です。* |






|  |
| --- |
|  |
| *ヘッドジャッジによる説明中です。あまりにプレイヤーが多いため一番手前のテーブルにはテーブルクロスが足りていないようです。* |

　参加人数はなんと1523人となり、日本マジック史上最大のグランプリとなりました。全10ラウンドの長丁場ですが、皆さん頑張ってください。  
 　それでは、いよいよ本戦が始まります。今後の更新をお楽しみに！


by Seo Asako
