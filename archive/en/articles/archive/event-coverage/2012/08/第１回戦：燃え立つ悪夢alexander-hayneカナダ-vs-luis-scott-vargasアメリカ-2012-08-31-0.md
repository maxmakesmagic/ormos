第１回戦：燃え立つ悪夢Alexander Hayne(カナダ) vs. Luis Scott-Vargas(アメリカ)
===========================================================



 Posted in **Event Coverage**
 on August 31, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_author.jpg)
By Wizards of the Coast












　最近の５年を通して、ルイス・スコット＝ヴァーガス（LSV）はプロツアーで最も堅実に結果を残し続けてきたプレイヤーのひとりだ。彼はどのトーナメントでも、そこで何が起こるかの深い理解を持って現れ、注意深く削りだされ中身を知り尽くしたデッキを操っている。


　LSVが持つこのゲームへの天性の熟達も重要な要素を占めるが、彼とチーム・ChannelFireballがトーナメントでの成功を最大化する超効率的なプレイテスト・マシーンとして機能しているという事実が、彼（と彼のチームメイト）を頂点へと確かに押し上げている。


　チーム・ChannelFireballの支配に触発され、アレクサンダー・ヘイン/Alexander Hayneを筆頭とするカナダを代表するプレイヤーたちは、国際的な影響力を持てるような、自分たちのチームを結成することを決めた。


　本格的にともに活動したのは数ヶ月のみであったにもかかわらず、彼らの努力はパスカル・メイナード/Pascal Maynardの複数回のグランプリトップ８、そして最も特筆すべき、ヘインがプロツアー・アヴァシンの帰還で優勝することができたという形ですでに、確かに報われた。

### ゲーム１


　マッチの前、プレイヤーたちは他のプレイヤーのデッキリストの情報を知る機会を与えられた。ヘインの黒緑デッキが《[適者生存](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%81%A9%E8%80%85%E7%94%9F%E5%AD%98)》《[繰り返す悪夢](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B9%B0%E3%82%8A%E8%BF%94%E3%81%99%E6%82%AA%E5%A4%A2)》《[グリセルブランド](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B0%E3%83%AA%E3%82%BB%E3%83%AB%E3%83%96%E3%83%A9%E3%83%B3%E3%83%89)》そして《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》をフィーチャーしたものであると知ると、LSVはこのマッチを取るためには彼の赤単バーンデッキで速やかなスタートを切る必要があることを理解した。

「このマッチアップはとても両極端だな。もし私が初手に《[ゴブリンの先達](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B4%E3%83%96%E3%83%AA%E3%83%B3%E3%81%AE%E5%85%88%E9%81%94)》か《[燃え上がる憤怒の祭殿](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%87%83%E3%81%88%E4%B8%8A%E3%81%8C%E3%82%8B%E6%86%A4%E6%80%92%E3%81%AE%E7%A5%AD%E6%AE%BF)》を持っているなら、そうでないのとはかなり違う。そして君が《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》を持っているなら、実に厳しいものになりそうだ…」


　LSVはダイスロールに勝ち、双方のプレイヤーが１回のマリガンを経て始めた。


　ヘインは《[ジョラーガの樹語り](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A7%E3%83%A9%E3%83%BC%E3%82%AC%E3%81%AE%E6%A8%B9%E8%AA%9E%E3%82%8A)》で入り、《[適者生存](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%81%A9%E8%80%85%E7%94%9F%E5%AD%98)》を続けた。一方のLSVは最初の数ターンを《[ミシュラのアンク](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%9F%E3%82%B7%E3%83%A5%E3%83%A9%E3%81%AE%E3%82%A2%E3%83%B3%E3%82%AF)》のキャストと《[ジョラーガの樹語り](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A7%E3%83%A9%E3%83%BC%E3%82%AC%E3%81%AE%E6%A8%B9%E8%AA%9E%E3%82%8A)》を除去する《[黒焦げ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%BB%92%E7%84%A6%E3%81%92)》に費やした。ヘインが第３ターンに（《[適者生存](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%81%A9%E8%80%85%E7%94%9F%E5%AD%98)》を伴う）《[繰り返す悪夢](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B9%B0%E3%82%8A%E8%BF%94%E3%81%99%E6%82%AA%E5%A4%A2)》をキャストすると、LSVは相手を焼き尽くすのにはあまり時間が残されていないことを知った。

|  |
| *LSVの戦いは劣勢から始まった。緊急にこの場を凌ぐ必要がある。* |

　《[地獄乗り](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%9C%B0%E7%8D%84%E4%B9%97%E3%82%8A)》はLSVに非常に良いダメージ源をもたらしたが、《[適者生存](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%81%A9%E8%80%85%E7%94%9F%E5%AD%98)》からの《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》がヘインに大きく時間を稼ぐことを助けた。LSVはヘインのライフを一時は６まで落とし込んだが、ヘインが《[繰り返す悪夢](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B9%B0%E3%82%8A%E8%BF%94%E3%81%99%E6%82%AA%E5%A4%A2)》で《[グリセルブランド](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B0%E3%83%AA%E3%82%BB%E3%83%AB%E3%83%96%E3%83%A9%E3%83%B3%E3%83%89)》を戻すまでに焼き尽くすほどの火力はとてもではないが持っておらず、このカナダ人プロが１ゲームをリードすることになった。


ヘイン 1-0 LSV


### ゲーム２


　第２ゲーム、LSVは初手をキープしたが、一方のヘインは５枚までマリガンを強いられた。


　LSVは《[ブリキ通りの悪党](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%96%E3%83%AA%E3%82%AD%E9%80%9A%E3%82%8A%E3%81%AE%E6%82%AA%E5%85%9A)》と《[ヴァルショクの難民](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%B4%E3%82%A1%E3%83%AB%E3%82%B7%E3%83%A7%E3%82%AF%E3%81%AE%E9%9B%A3%E6%B0%91)》で素早くヘインをおびやかすのだが、《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》はLSVの攻撃陣を完璧に抑えこむのに十分であった。


　１ターン後に《[繰り返す悪夢](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B9%B0%E3%82%8A%E8%BF%94%E3%81%99%E6%82%AA%E5%A4%A2)》がヘインのもとにやってきた。今回は《[適者生存](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%81%A9%E8%80%85%E7%94%9F%E5%AD%98)》を持っていないが、それでも《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》を墓地から即座に戻す手段を提供した。


　《[渋面の溶岩使い](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%B8%8B%E9%9D%A2%E3%81%AE%E6%BA%B6%E5%B2%A9%E4%BD%BF%E3%81%84)》、《[尖塔の源獣](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B0%96%E5%A1%94%E3%81%AE%E6%BA%90%E7%8D%A3)》、《[ぐらつく峰](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%81%90%E3%82%89%E3%81%A4%E3%81%8F%E5%B3%B0)》によってLSVはいくらかのダメージを通すことができたが、ヘインは（当然ながら）《[繰り返す悪夢](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B9%B0%E3%82%8A%E8%BF%94%E3%81%99%E6%82%AA%E5%A4%A2)》で《[樹上の村](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A8%B9%E4%B8%8A%E3%81%AE%E6%9D%91)》を《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》に換え、さらなる時間を稼いだ。




|  |
| --- |
|  |
| *《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》は、赤単のプレイヤーにとってしばしば「繰り返す悪夢」となる。* |

　LSVは最終的にヘインをライフ５まで追い詰めることができたが、その任務が完了する前に《[虐殺のワーム](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%99%90%E6%AE%BA%E3%81%AE%E3%83%AF%E3%83%BC%E3%83%A0)》が彼のクリーチャーをすべて吹き飛ばし、最後のダメージを与える火力呪文が今すぐ必要な状態に追い込まれた。


　《[悪鬼の血脈、ティボルト](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%82%AA%E9%AC%BC%E3%81%AE%E8%A1%80%E8%84%88%E3%80%81%E3%83%86%E3%82%A3%E3%83%9C%E3%83%AB%E3%83%88)》はLSVに決め手を探して掘り進めることを助けたが、《[グリセルブランド](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B0%E3%83%AA%E3%82%BB%E3%83%AB%E3%83%96%E3%83%A9%E3%83%B3%E3%83%89)》によって、LSVが第３ゲームに進むための奇跡的なドローへの猶予はわずか１ターンとなってしまった。


　LSVは土地５枚の状態で《[黒焦げ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%BB%92%E7%84%A6%E3%81%92)》と《[忌むべき者のかがり火](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BF%8C%E3%82%80%E3%81%B9%E3%81%8D%E8%80%85%E3%81%AE%E3%81%8B%E3%81%8C%E3%82%8A%E7%81%AB)》を引きはしたが、ヘインのライフは５で彼を倒すにはわずかに足りず、最後には倒れ伏したのだった。


ヘイン 2-0 LSV


 


(Tr. Yusuke Yoshikawa)
