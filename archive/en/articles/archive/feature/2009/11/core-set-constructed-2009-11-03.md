Looking for a fun **Magic Online** format? Try Core Set Constructed. The only sets allowed are—you guessed it—core sets. Cpage01 took the age-old strategy of blue-white control to a top finish with the concoction assembled above. How will you beat that strategy using only cards found in sets like ***Magic** 2010*, *Tenth Edition*, and *Ninth Edition*?
