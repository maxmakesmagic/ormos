*Eighth Edition* Rollout - *Legions*
====================================



 Posted in **Feature**
 on June 9, 2003 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_author.jpg)
By Wizards of the Coast












*Wizards are now the marquee race for blue.*


### *[Legions](http://archive.wizards.com/default.asp?x=magic/expansion/legions)* Information:




|  |  |  |
| --- | --- | --- |
| **Released:** February 2003**Design Team:** Mike Elliott (lead), Mike Donais**Development Team:** William Jockusch (lead), Elaine Chase, Mike Donais, Worth Wollpert**Art Director:** Jeremy Cranford & Dana Knutson |  | **New Mechanics:** amplify, double strike, provoke, morph triggered abilities**Factoid:** *Legions* was all creatures. No instants, no sorceries, nothing. Just 145 creatures. |

### Cards in *Eighth Edition* that originated in *Legions*:




|  |  |  |
| --- | --- | --- |
| **[Enormous Baloth](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Enormous+Baloth)** (S) | **[Fugitive Wizard](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Fugitive+Wizard)** | **[Merchant of Secrets](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Merchant+of+Secrets)** |

**Bold** indicates a card that has never appeared in a Core Set before.  

(S) indicates a card that will exist only in the Core Game.
