Round 13: 鷲見 和男(埼玉) vs. 吉田 新太郎(岡山)
==================================



 Posted in **Event Coverage**
 on June 24, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_authorpic_larger.jpg)
By Wizards of the Coast











　日曜日も正午を過ぎて、グランプリも終盤戦。


　いつもならばそろそろプレイオフ進出者が決まってる時間ではあるが、2日目を7回戦行う今回の横浜では未だ折り返し地点。   
 　この時点で4番テーブルに座る1敗の鷲見と1敗1分けの吉田ですら賞金が確定している訳ではないのだ。 


　この位置というのはトーナメントの中で最も苦しいのではないかとさえ思う。   
 　一方で見る側としてはこれほど面白いマッチもないだろう。   
 　第13ラウンドは勝てばトップ8まであと1歩、最悪でも賞金確定。負ければ再び馬群に飲み込まれるこの一戦をお届けするとしよう。 


![R13_yoshida_VS_washimi.jpg](https://media.magic.wizards.com/image_legacy_migration/images/magic/daily/eventsjp/gpyok12/R13_yoshida_VS_wasimi.jpg)


### ゲーム１


　先攻を取った鷲見がキープ、一方で吉田がマリガン、続けてダブルマリガン。


　《[寺院の庭](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89093)》から《[極楽鳥](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》という展開に対して、《[森](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE)》からの《[古きものの活性](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%A4%E3%81%8D%E3%82%82%E3%81%AE%E3%81%AE%E6%B4%BB%E6%80%A7)》。   
 　このカードを使うデッキタイプは環境に1つしか存在しない、ウルザトロンだ。 


　この時点でほとんどお互いにデッキが分かっているようなものだが、続けての鷲見の《[聖なる鋳造所](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89066)》アンタップインとファイレクシア・マナ、4点ライフ支払いの上で《[出産の殻](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》を設置で、吉田に明確なプレッシャーと同時にデッキの正体を現す。 


　クリーチャーを軸にした循環デッキ。《[シルヴォクののけ者、メリーラ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B7%E3%83%AB%E3%83%B4%E3%82%A9%E3%82%AF%E3%81%AE%E3%81%AE%E3%81%91%E8%80%85%E3%80%81%E3%83%A1%E3%83%AA%E3%83%BC%E3%83%A9)》+《[残忍なレッドキャップ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%AE%8B%E5%BF%8D%E3%81%AA%E3%83%AC%E3%83%83%E3%83%89%E3%82%AD%E3%83%A3%E3%83%83%E3%83%97)》、《[鏡割りのキキジキ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%8F%A1%E5%89%B2%E3%82%8A%E3%81%AE%E3%82%AD%E3%82%AD%E3%82%B8%E3%82%AD)》+《[修復の天使](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》などなどなど、どのような形にせよ早ければ2ターン。遅くとも3ターン後には致死ダメージに至ってしまうのは想像に難くない。 


　《[大祖始の遺産](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A4%A7%E7%A5%96%E5%A7%8B%E3%81%AE%E9%81%BA%E7%94%A3)》で《[台所の嫌がらせ屋](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》の頑強への睨みを効かせるという渋いプレイで1〜2ターンを稼ぎ出してウルザトロンを揃えることには成功した吉田であったが、鷲見の手から召喚された《[鏡割りのキキジキ](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%8F%A1%E5%89%B2%E3%82%8A%E3%81%AE%E3%82%AD%E3%82%AD%E3%82%B8%E3%82%AD)》に対してどうすることもできず、前のターンに《[台所の嫌がらせ屋](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》から孵化していた《[修復の天使](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》とコンボ達成となった。 


*鷲見 1-0 吉田*



### ゲーム２




|  |
| --- |
|  |
| *吉田 新太郎* |

　吉田が7枚、鷲見が6枚からでのスタート。


　土地が思うように伸びずマナの展開をクリーチャーに託す鷲見に対して、ウルザトロンを揃えながら《[差し戻し](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=87919)》や《[紅蓮地獄](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B4%85%E8%93%AE%E5%9C%B0%E7%8D%84)》で対応し、今回が吉田が完全にペースを掴んでいる。   
 　鷲見にとって、救いは吉田がまだゲームを決める何かを引いてはいないということではあるのだが… 


　キャントリップ呪文でデッキを掘り進めた吉田が《[解放された者、カーン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A7%A3%E6%94%BE%E3%81%95%E3%82%8C%E3%81%9F%E8%80%85%E3%80%81%E3%82%AB%E3%83%BC%E3%83%B3)》を引き当てたことにより決着は最終ゲームへ。 


*鷲見 1-1 吉田*



### ゲーム３




|  |
| --- |
|  |
| *鷲見 和男* |

　再びの先攻の鷲見。


　《[極楽鳥](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》から《[大爆発の魔道士](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A4%A7%E7%88%86%E7%99%BA%E3%81%AE%E9%AD%94%E9%81%93%E5%A3%AB)》という対トロンに理想的なスタート。   
 　吉田は《[ウルザの鉱山](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83314)》、《[彩色の宝球](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BD%A9%E8%89%B2%E3%81%AE%E5%AE%9D%E7%90%83)》。さらに《[ウルザの塔](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83316)》から《[森の占術](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE%E3%81%AE%E5%8D%A0%E8%A1%93)》で《[ウルザの魔力炉](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83315)》をサーチ。マリガンしての6枚にしては非常に良い動きではあったが、トロン成立が阻まれている状況ではやや苦しいか。 


　返して鷲見は《[召喚の調べ](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89064)》X=2で《[根の壁](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A0%B9%E3%81%AE%E5%A3%81)》を用意して、再びのX=4《[召喚の調べ](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89064)》。   
 　虎の子である《[召喚の調べ](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89064)》を使ってまでマナを増やすことを選んだ理由は、もちろん《[なだれ乗り](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%81%AA%E3%81%A0%E3%82%8C%E4%B9%97%E3%82%8A)》だ。続けて《[修復の天使](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》の明滅で土地破壊のただ乗り継続。吉田の土地の3枚目を並ばせない。   
 　おそらく活躍の機会はないであろう《[村の鐘鳴らし](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%9D%91%E3%81%AE%E9%90%98%E9%B3%B4%E3%82%89%E3%81%97)》までもを戦線に投入して一気にゲームを決めようとする鷲見。 


　だが、3枚ものウルザ土地を割られながら、吉田も考えうる限り理想ともいえる対応で逆襲を始める。   
 　《[修復の天使](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》に対して《[焼却](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%84%BC%E5%8D%B4)》、追加の《[大爆発の魔道士](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A4%A7%E7%88%86%E7%99%BA%E3%81%AE%E9%AD%94%E9%81%93%E5%A3%AB)》を巻き込む、《[森の占術](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE%E3%81%AE%E5%8D%A0%E8%A1%93)》で《[蒸気孔](http://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=96923)》からの《[紅蓮地獄](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B4%85%E8%93%AE%E5%9C%B0%E7%8D%84)》。   
 　息が完全に切れてしまった鷲見に対し、ライフを5残して踏みとどまることに成功した。 


　土地ではない2枚のカードを持っている吉田に対し、0枚の鷲見。   
 　あと半歩、なんとかしたい鷲見だが後続を引けない。だが吉田も土地を引けない。 


　その間に、《[村の鐘鳴らし](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%9D%91%E3%81%AE%E9%90%98%E9%B3%B4%E3%82%89%E3%81%97)》が最後の5点を5ターンかけて削りきってしまった。 


　吉田は唱えられない2枚の《[紅蓮地獄](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B4%85%E8%93%AE%E5%9C%B0%E7%8D%84)》と4枚の《[解放された者、カーン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A7%A3%E6%94%BE%E3%81%95%E3%82%8C%E3%81%9F%E8%80%85%E3%80%81%E3%82%AB%E3%83%BC%E3%83%B3)》を公開して投了。   
 　鷲見が限りなく貴重な1勝を加えて、吉田はトップ8へは負けられない状態となって第14ラウンドへ進むこととなった。 


*鷲見 2-1 吉田*



  



By 中村 修平
