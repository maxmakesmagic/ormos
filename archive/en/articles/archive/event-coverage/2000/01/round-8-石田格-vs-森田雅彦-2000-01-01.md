Round 8: 石田格 VS. 森田雅彦
=====================



 Posted in **Event Coverage**







![](https://media.magic.wizards.com/styles/auth_small/public/generic-avatar-150_612.png)
By 藤枝勇











![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpsen01/948.JPG)本日 3 回目の登場、石田本人は嘆いていましたが対戦が良いのだから仕方なし。  

対戦相手は GP Finalist 2 回 日本代表経験も有る西の雄、森田雅彦これを Feature しない理由無し。  

今大会はいわゆる有名所が多数上位にいるためレポートするほうも楽しみで仕方ありません、まぁ御託は無しにして東西が誇る雄同士の対戦をお送りいたします。


### Game 1


先攻石田　後攻森マリガン  

Game 1 は一方的な展開であった、 2 ターン目に出た石田の《 Wild Mongrel / 野生の雑種犬 》が殴る殴る殴る。  

手札にあった《 Krovikan Horror 》 \* 2 《 Squee, Gblin Nabobo / ゴブリンの太守スクイー 》 \* 2 を捨て 3 ターン目から 6 / 6 の化け物が完成した。  

森田も《 Pernicious Deed / 破滅的な行為 》で流すもののそのターン中《 Zombie Infestation / ゾンビの横行 》が出てくる有様･･･。  

一方的なアタックにより森田圧殺死。


石田 1 - 0 森田


### Game 2


先攻森田　後攻石田  

森田は《 Duress / 強迫 》を撃つと《 Firestorm / 炎の嵐 》 2 《 Zombie Infestation / ゾンビの横行 》 2 《Null Rod / 無のロッド 》《 Swamp / 沼 》 《 Squee, Gblin Nabobo / ゴブリンの太守スクイー 》から《 Firestorm / 炎の嵐 》を抜く。  

石田ランドを引かないときついところだったが無事《 Taiga 》を引き《 Zombie Infestation / ゾンビの横行 》をキャストできた。  

森田《 Spike Feeder / スパイクの飼育係 》をキャストしアタックしたいところだが《 Firestorm / 炎の嵐 》と《 Zombie Infestation / ゾンビの横行 》のせいでできないでいる。逆に石田は《 Firestorm / 炎の嵐 》のおかげで攻勢に回ることができたが森の《 Spiritmonger / 魂売り 》の登場によりきつい展開となる。  

立て続けに攻める森《 Dust Bowl / 黄塵地帯 》を出し石田の 2 枚しか無いランドを攻め止めとばかりに《 Pernicious Deed / 破滅的な行為 》を張り、ブロックができない石田を圧殺する。


石田 1 - 1 森田


### Game 3


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpsen01/949.JPG)共にマリガンなし先攻石田は《 Duress / 強迫 》で《 Duress / 強迫 》と《 Vampiric Tutor / 吸血の教示者 》の 2 択で将来《 Pernicious Deed / 破滅的な行為 》を持ってこられると困るので《 Vampiric Tutor / 吸血の教示者 》を抜く。  

森田も《 Duress / 強迫 》を打ち《 Perish / 非業の死 》を抜き《 Birds of Paradise / 極楽鳥 》を出す。  

石田が動かないところを見、森田は《 Yavimaya Elder / ヤヴィマヤの古老 》を出し攻勢に回りたいところだったがが《 Birds of Paradise / 極楽鳥 》と共に《 Firestorm / 炎の嵐 》の餌食となる。  

返しに石田は《 Wild Mongrel / 野生の雑種犬 》を呼び攻撃開始  

石田の 2 体に増えた《 Wild Mongrel / 野生の雑種犬 》 を止めようと置いておいた《 Treetop Village / 樹上の村 》をクリーチャーにしようとしたところでまたしても《 Firestorm / 炎の嵐 》！  

森田の場には何もいなくなり《 Firestorm / 炎の嵐 》と《 Wild Mongrel / 野生の雑種犬 》のダメージが入り押し切る形になるかと思ったが、森田は返しのターンに《 Pernicious Deed / 破滅的な行為 》を張り石田の場を何も無い状態まで戻す。  

石田は今日のデッキの中で一枚 Good Card を挙げてくださいという質問にこう答えた「《 Firestorm / 炎の嵐 》」と、そうこのデュエルは最初から最後まで石田の《 Firestorm / 炎の嵐 》がまさに火を噴いた森田が《 Duress / 強迫 》を撃つと石田はレスポンスで《 Vampiric Tutor / 吸血の教示者 》をキャスト石田のハンドの中に《 Gemstone Mine / 宝石鉱山 》を発見すると、諦めたようにターンを返した。  

そして最後《 Vampiric Tutor / 吸血の教示者 》で積まれた《 Firestorm / 炎の嵐 》を 3 点で放ち今日 3 回目となる Feature Match を勝利で終わらせた。


**Final Results :** 石田 2 - 1 森田
