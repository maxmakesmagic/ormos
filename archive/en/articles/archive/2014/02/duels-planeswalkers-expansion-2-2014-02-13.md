|  |  |

| September 1, 2010 on PC |
| November 23, 2010 on Playstation 3 |

 


Features:


* Three new playable decks, including Nicol Bolas's "Eons of Evil" Planeswalker deck
* Three new unlockable cards for each of the eight original decks plus the three decks from expansion 1
* Five new puzzling challenge levels
* Three new achievements
* Host migration for multiplayer
* Avatar attire awards (on Xbox LIVE only)

 


### Achievements




|  |  |  |  |
| --- | --- | --- | --- |
|  | **Name** | **Description** | **Points** |
|  | Last One Standing | Win a 3- or 4-player Xbox LIVE match | 15 |
|  | Overflowing Harvest | Put 8 lands onto the battlefield in one turn | 15 |
|  | Everything At Stake | Defeat Sorin Markov in the single-player campaign (Expansion 2) | 20 |
