準々決勝: 牛島 正規(東京) vs. 山口 聡史(愛知)
=============================



 Posted in **Event Coverage**
 on June 22, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_author.jpg)
By Wizards of the Coast











　今回、とあるコミュニティの作成してきた《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》デッキが、トップ8の席のうちの半分、4席を埋めた。それが「ファミコンくん2号店」のプレイヤーを中心とした、牛島も参加するコミュニティである。 



|  |
| --- |
|  |
| *牛島 正規* |

　他にトップ8に名前を連ねた角岡、永見、四本らと共に調整を進めてきた牛島としては、できるならば決勝まで駆け上がって仲間と戦いたい。その想いを叶えるため、準々決勝を戦う。 

　予選1位通過の山口は愛知で練習を重ねてきたプレイヤー。 　これまでもバブルマッチで涙を呑んだことは多かったが、今回は念願のトップ8だ。愛知のチームで調整を重ね、今回は3人でほぼ同じジャンドを使用しトップ8まで駆け上がってきた。チームメンバーのうちもう1人も2日目の上位に残っており、デッキの完成度の高さが伺える。 

　二人とも緊張している様子でお互いのデッキをシャッフル。 　殻 vs ジャンド、環境を代表する強力デッキ同士の対決となった準々決勝。 　果たしてどうなるのか。 

### ゲーム１

![QF_ushijima_VS_yamaguchi.jpg](https://media.magic.wizards.com/image_legacy_migration/images/magic/daily/eventsjp/gpyok12/QF_ushijima_VS_yamaguchi.jpg)

　1位通過の山口が先攻を宣言。少し考えてキープした。牛島はマリガンを選択。一応ながら展開できる手札だったが、クリーチャーしか居ないのでは除去の豊富なジャンドに対しては心許ないという判断だろう。牛島は6枚をキープ。 

　この手札はすぐに山口の《[コジレックの審問](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B3%E3%82%B8%E3%83%AC%E3%83%83%E3%82%AF%E3%81%AE%E5%AF%A9%E5%95%8F)》によって公開される。 

* 《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》
* 《[永遠の証人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%B0%B8%E9%81%A0%E3%81%AE%E8%A8%BC%E4%BA%BA)》
* 《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》
* 《[召喚の調べ](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89064)》
* 《[ガヴォニーの居住区](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AC%E3%83%B4%E3%82%A9%E3%83%8B%E3%83%BC%E3%81%AE%E5%B1%85%E4%BD%8F%E5%8C%BA)》
* 《[森](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE)》

という手札から、《[永遠の証人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%B0%B8%E9%81%A0%E3%81%AE%E8%A8%BC%E4%BA%BA)》が選択される。 

　《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》が生き残れば順調な展開ができそうな手札だ。そう、生き残れば。この《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》には当然のように《[稲妻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%A8%B2%E5%A6%BB)》が見舞われる。  
 　土地が《[ガヴォニーの居住区](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AC%E3%83%B4%E3%82%A9%E3%83%8B%E3%83%BC%E3%81%AE%E5%B1%85%E4%BD%8F%E5%8C%BA)》と《[森](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE)》の2枚で完全にストップした牛島。山口も土地こそ2枚で一旦は止まるものの、《[闇の腹心](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83771)》で土地を手に入れ、《[タルモゴイフ](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=136142)》を追加。 

　結局3枚目の土地を引けなかった牛島は、そのまま山口のクリーチャー陣を止めることをできずに無念の投了となった。 

*山口 1-0 牛島*


　《[呪文滑り](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%91%AA%E6%96%87%E6%BB%91%E3%82%8A)》《[忌むべき者のかがり火](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BF%8C%E3%82%80%E3%81%B9%E3%81%8D%E8%80%85%E3%81%AE%E3%81%8B%E3%81%8C%E3%82%8A%E7%81%AB)》などを追加する牛島。  
 　《[古えの遺恨](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%A4%E3%81%88%E3%81%AE%E9%81%BA%E6%81%A8)》などのアーティファクト破壊や《[墓掘りの檻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A2%93%E6%8E%98%E3%82%8A%E3%81%AE%E6%AA%BB)》を追加する山口。 

　お互い想定し、十分に練習しているであろうマッチアップ、相応にサイドボード枠が確保されている。 

### ゲーム２

　ゲーム1に続いて重くてマナクリーチャーの無い手札が来た牛島、マリガンを宣言。 

　山口は《[思考囲い](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%80%9D%E8%80%83%E5%9B%B2%E3%81%84)》の様子見から入れるが緑マナが無いハンドをキープ。  
 　牛島はマリガン後に《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》と《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》《[幻影の像](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B9%BB%E5%BD%B1%E3%81%AE%E5%83%8F)》《[森](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE)》《[森](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE)》《[平地](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B9%B3%E5%9C%B0)》という手札をキープした。 

　牛島の《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》と、山口のそれに対する《[稲妻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%A8%B2%E5%A6%BB)》からゲームがスタート。2ターン目に《[思考囲い](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%80%9D%E8%80%83%E5%9B%B2%E3%81%84)》を打った山口は《[幻影の像](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B9%BB%E5%BD%B1%E3%81%AE%E5%83%8F)》を捨てさせ、《[樹上の村](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=243455)》で緑マナを確保。 



|  |
| --- |
|  |
| *山口 聡史* |

　返しで《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》が出てくるのは予定通り、これに山口は《[墓掘りの檻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A2%93%E6%8E%98%E3%82%8A%E3%81%AE%E6%AA%BB)》で対処。さらに引いた《[樹上の村](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=243455)》を設置。 

　《[台所の嫌がらせ屋](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%B0%E6%89%80%E3%81%AE%E5%AB%8C%E3%81%8C%E3%82%89%E3%81%9B%E5%B1%8B)》を展開した山口は、牛島の《[根の壁](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=106662)》に《[血編み髪のエルフ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A1%80%E7%B7%A8%E3%81%BF%E9%AB%AA%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》からの《[終止](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B5%82%E6%AD%A2)》で対処！ 

　マナもブロッカーも減らされたうえで、6点のダメージを受けてしまった牛島。ここで展開するのは、現状は《[墓掘りの檻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A2%93%E6%8E%98%E3%82%8A%E3%81%AE%E6%AA%BB)》のせいでただの4/3飛行となった《[目覚ましヒバリ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%9B%AE%E8%A6%9A%E3%81%BE%E3%81%97%E3%83%92%E3%83%90%E3%83%AA)》とかみ合わない。 

　墓地からクリーチャーを戻せないことを確認した山口は、そのまま《[樹上の村](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=243455)》もクリーチャー化し3体のパワー3のクリーチャーをレッドゾーンへ。《[血編み髪のエルフ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A1%80%E7%B7%A8%E3%81%BF%E9%AB%AA%E3%81%AE%E3%82%A8%E3%83%AB%E3%83%95)》はブロックされるも、通った《[樹上の村](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=243455)》に《[ジャンドの魔除け](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B8%E3%83%A3%E3%83%B3%E3%83%89%E3%81%AE%E9%AD%94%E9%99%A4%E3%81%91)》で+1/+1カウンターを2つ乗せ、合計8点のダメージが牛島のライフを削りきった。 

*山口 2-0 牛島*

　山口 聡史、準決勝進出！

  


By Masami Kaneko
