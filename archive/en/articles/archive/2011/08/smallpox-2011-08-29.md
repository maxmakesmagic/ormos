As the first Modern deck lists start rolling in, we're beginning to see the format as it's shaping up for Pro Tour Philadelphia. This deck, played in a recent **Magic Online** event, features an old favorite newly reprinted in Standard: [Smallpox](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Smallpox). It's a good bet [Smallpox](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Smallpox) is useful against TwelvePost decks abusing [Cloudpost](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Cloudpost) to cast fourth-turn [Emrakul, the Aeons Torn](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Emrakul%2C+the+Aeons+Torn)! For more Modern deck lists, check out [the **Magic Online** page](/en/articles/archive/2009-06-29).
