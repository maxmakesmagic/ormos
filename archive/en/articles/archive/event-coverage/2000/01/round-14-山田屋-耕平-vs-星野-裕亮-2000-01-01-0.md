Round 14: 山田屋 耕平 vs 星野 裕亮
=========================



 Posted in **Event Coverage**







![](https://media.magic.wizards.com/styles/auth_small/public/generic-avatar-150_144.png)
By 藤枝 勇











![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/890.jpg)星野、山田屋は共に DCI Japan でマネードラフトに勤しむ仲間同士である。  

星野においては Team CMD として PT NY にも参加、後一歩で 2 日目に残れる所までいったなかなかのやり手だ。  

意外なことに山田屋は今回がグランプリ参加 2 回目だという決勝に抜けるようならなかなかの快挙アマチュア賞金も出るであろう。  

CMD どうしのこのマッチ勝てば決勝とは信じられないような雰囲気の中行われた。


### Game 1


山田屋 ( WU )、星野 ( GB ) 共にマリガンは無し、序盤は《 Pilgrim of Justice / 正義の巡礼者 》が出るだけのゆっくりとした展開。  

星野がクリーチャーを引けない間に山田屋は《 Treetop Sentinel / 樹上の歩哨 》《 Aven Archer / エイヴンの射手 》と召喚し場を固め飛行でゆっくりと殴り始める。  

星野も 8 ターン目にようやく《 Nantuko Disciple / ナントゥーコの信奉者 》《 Krosan Avenger / クローサの報復者 》を引くが飛行を止める手段が無く投了。


山田屋 1 - 0 星野


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/891.jpg)### Game 2


Side Board で星野は山田屋の飛行戦隊を除去するために赤を加えて 3 色に変化、事故が起きる可能性は高まるが苦肉の選択だろう。  

共にマリガンは無く星野が 3 ターン目に《 Krosan Avenger / クローサの報復者 》を出し攻撃を開始、山田屋もブロックを試みるのだが星野の 4 枚入ってる《 Muscle Burst / 筋力急伸 》の前に阻まれる。  

ダメージを上げようと星野が《 Springing Tiger / 飛びかかる虎 》を出した所で山田屋の《 Kirtar's Wrath / カーターの怒り 》が炸裂し、場がまっさらな状態に戻る。  

そこで星野の《 Price of Glory / 栄光の代価 》が張られた、ぱっと見た所なぜ投入されたか疑問だったが山田屋の《 Second Thoughts / 考え直し 》や《 Embolden / 励まし 》対策のようだ。  

星野はその後《 Braids, Cabal Minion / 陰謀団の先手ブレイズ 》《 Crypt Creeper / 墓所を歩くもの 》を出して、攻撃力を上げていくが、依然飛行が止まらず山田屋の《 Aven Fisher / エイヴンの魚捕り 》との殴り合いになり後続を引かず《 Embolden / 励まし 》で凌いでいた山田屋はダメージレースに負けこの Game を落とす。


山田屋 1 - 1 星野


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/892.jpg)### Game 3


後攻星野がマリガンするも土地が 5 枚というハンドに顔をしかめながらもスタート。  

だが山田屋は UW という比較的遅いデッキのため《 Pilgrim of Justice / 正義の巡礼者 》しかだせず展開の遅い星野を攻めきれない。  

その間に《 Werebear / 熊人間 》《 Krosan Avenger / クローサの報復者 》を引き星野が先に殴れる体制になるかに見えたが、p:cre の登場により攻撃の手が止まってしまう。  

何とかして《 Beloved Chaplain / 敬愛される司祭 》を除去したい星野だが手札には《 Thermal Blast / 炎熱の突風 》しか無く、引いてきた《 Afflict / 苦悩 》も《 Ather Burst / 霊気の噴出 》によって回避されてしまう。  

その間に《 Aven Fisher / エイヴンの魚捕り 》《 Treetop Sentinel / 樹上の歩哨 》を引き攻撃力を上げてく山田屋、星野は Game 1 と同じく飛行戦隊が止められず決勝進出を逃してしまった。


山田屋 2 - 1 星野


山田屋 Win
