True Devotion
=============



 Posted in [ARTICLES](/en/articles)
 on March 11, 2014 









Mono-Blue Devotion is dead.


 



![Ephara, God of the Polis](http://gatherer.wizards.com/Handlers/Image.ashx?size=small&type=card&name=Ephara, God of the Polis&options=) If you've been paying attention, Mono-Blue Devotion hasn't been putting up great finishes ever since *Born of the Gods* was released. Nearly every deck out there received some nice upgrades except for Mono-Blue. As a result, Mono-Blue stopped putting up the numbers that it did pre-BNG.
What the archetype needed is a boost that would allow it to keep up with the other archetypes. As DaveLombardoJr discovered, splashing white for [Ephara, God of the Polis](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Ephara%2C+God+of+the+Polis) and [Detention Sphere](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Detention+Sphere) is probably what the deck needed. Ephara allows the deck to clean up the [Bident of Thassa](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Bident+of+Thassa)/[Jace, Architect of Thought](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Jace%2C+Architect+of+Thought) splits the Mono-Blue versions used to play. Similarly, [Detention Sphere](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Detention+Sphere) cleans up the [Cyclonic Rift](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Cyclonic+Rift)/[Rapid Hybridization](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Rapid+Hybridization) slot. It also deals with multiple [Pack Rat](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Pack+Rat)s, which was always an issue.


Most of the time, all a deck needs is a little fine tuning in order to be successful. Long live WU Devotion?
