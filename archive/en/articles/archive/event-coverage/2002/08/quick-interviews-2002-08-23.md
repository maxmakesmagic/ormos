Quick Interviews
================



 Posted in **Event Coverage**
 on August 23, 2002 






![](https://web.archive.org/web/20211024110319im_/https://magic.wizards.com/sites/all/themes/wiz_mtg/images/global/generic-avatar-150.png)
By Sideboard Staff











**"What was your toughest match-up/hardest opponent on Day 1?"**



|  |  |  |
| --- | --- | --- |
|  |  |  |
| **Masahiro Kuroda:**"Reiji Andou. My last match, against Akira Asahara, was also really rough." | **Osamu Fujita:**"They were all hard, but I was ready to give up against Masahiko Morita." | **Kunisato Ebie:**"Fumiyasu Morikawa, one of my friends from Sapporo." |

**"How was your trip to Sydney?"**



|  |  |  |
| --- | --- | --- |
|  |  |  |
| **Kentarou Yuasa:**"I really enjoyed it. Australia is a great place." | **Jin Okamoto:**"I appreciate the fact that I can escape the summer heat by playing Magic, first in Sydney, now in Sapporo." | **Yuuki Murakami:**"It was my first time overseas. I had a really good time." |
