September 1 2008 DCI Banned and Restricted List Announcement
============================================================



 Posted in **Feature**
 on September 1, 2008 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_authorpic_larger.jpg)
By Wizards of the Coast











**Announcement Date:** September 1, 2008  
**Effective Date:** September 20, 2008  
**Magic Online Effective Date:** September 17, 2008

**Extended**  
[Sensei's Divining Top](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Sensei%27s+Divining+Top) is banned

**Legacy**  
[Time Vault](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Time+Vault) is banned\*

**Vintage**  
[Time Vault](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Time+Vault) is restricted\*  
[Chrome Mox](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Chrome+Mox) is no longer restricted  
[Dream Halls](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Dream+Halls) is no longer restricted  
[Mox Diamond](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Mox+Diamond) is no longer restricted  
[Personal Tutor](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Personal+Tutor) is no longer restricted  
[Time Spiral](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Time+Spiral) is no longer restricted

\*[Time Vault](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Time+Vault)'s power-level errata will be removed in conjunction with the next Oracle update on September 26. The Update Bulletin on September 24 will contain [Time Vault](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Time+Vault)'s new Oracle text and an explanation of the change.

**Standard, *Lorwyn-Shadowmoor* Block Constructed**  
 No changes

### Magic Online Formats

**Vanguard Avatar Rotation**  
 Each Vanguard avatar will now be available only for play formats in which its corresponding card is legal, disregarding Banned and Restricted Lists. The exception is the basic avatars that come with the **Magic Online** client, which will remain legal in all formats.

Players can still use any avatars they own to represent them in games that donâ€™t use the Vanguard rules.

**Online Classic**  
(and Vanguard Classic)  
[Imperial Seal](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Imperial+Seal) is restricted  
[Mana Crypt](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Mana+Crypt) is restricted

**100 Card Singleton**  
[Grindstone](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Grindstone) is banned\*\*  
[Lion's Eye Diamond](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Lion%27s+Eye+Diamond) is banned  
[Protean Hulk](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Protean+Hulk) is banned  
[Rune-Tail, Kitsune Ascendant](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Rune-Tail%2C+Kitsune+Ascendant) is no longer banned  
[Test of Endurance](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Test+of+Endurance) is no longer banned

\*\*[Grindstone](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Grindstone) is not currently available online but is being preemptively banned.

**Freeform, Prismatic Singleton, Tribal Wars Classic, Tribal Wars Standard**  
 No changes

For an explanation of this announcement, read the Latest Developments column on Friday, September 5. For the complete list of all banned and restricted cards by format, click [here](http://archive.wizards.com/Magic/Magazine/Article.aspx?x=judge/resources/banned).
