The Captain and the Soldier
===========================



 Posted in **Arcana**
 on September 21, 2012 






![](https://web.archive.org/web/20211024110319im_/https://magic.wizards.com/sites/all/themes/wiz_mtg/images/global/generic-avatar-150.png)
By Monty Ashley











Earlier this week, we looked at [the *Return to Ravnica* tokens](/en/node/646961) along with some of the cards that create them. For example, Precinct Captain makes this Soldier token:



|  |  |
| --- | --- |
| Precinct Captain | Return to Ravnica Soldier Token |

Because this is Arcana, you've probably already realized there's something interesting about these two cards. Right? Well, how about now? Because there is! They're actually standing back to back, so the Precinct Captain is in the background of the Soldier token, and vice versa!

![](https://media.magic.wizards.com/image_legacy_migration/images/magic/daily/arcana/1065_captain.jpg)Precinct Captain | Art by [Steve Prescott](http://gatherer.wizards.com/Pages/Search/Default.aspx?output=spoiler&method=visual&action=advanced&artist=%5B%22Steve%20Prescott%22%5D)

![](https://media.magic.wizards.com/image_legacy_migration/images/magic/daily/arcana/1065_soldier.jpg)*Return to Ravnica* Soldier Token | Art by [Steve Prescott](http://gatherer.wizards.com/Pages/Search/Default.aspx?output=spoiler&method=visual&action=advanced&artist=%5B%22Steve%20Prescott%22%5D)
