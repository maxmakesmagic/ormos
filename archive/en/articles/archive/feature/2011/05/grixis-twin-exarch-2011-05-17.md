The first large event featuring the latest set took place this past weekend! StarCityGames.com hosted a Standard Open in Orlando, FL, and Michael Strunk went all the way to the finals with this innovative combo deck. Simply stick [Splinter Twin](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Splinter+Twin) onto a copy of [Deceiver Exarch](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Deceiver+Exarch) and you can make an arbitrarily large number of copies of the 1/4 creature, untapping the original each time. It also plays as a more traditional blue-black-red control deck. For more information on the event, read the [coverage](http://www.starcitygames.com/php/news/expandnews.php?Article=21821) from StarCityGames.com.
