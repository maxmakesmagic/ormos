Announcing the Magic 2014 Core Set
==================================



 Posted in **Daily Deck**
 on January 7, 2013 






![](https://web.archive.org/web/20211024110319im_/https://magic.wizards.com/sites/all/themes/wiz_mtg/images/global/generic-avatar-150.png)
By Monty Ashley











![Magic 2014 Core Set Logo](https://media.wizards.com/images/magic/daily/arcana/1141_m14_logo_0j44kadtgt.jpg)***Ignite Your Spark***

 (Not available in all languages) |
