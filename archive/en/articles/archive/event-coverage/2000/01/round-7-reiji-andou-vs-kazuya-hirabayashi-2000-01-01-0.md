Round 7: Reiji Andou Vs. Kazuya Hirabayashi
===========================================



 Posted in **Event Coverage**







![](https://media.magic.wizards.com/styles/auth_small/public/generic-avatar-150_493.png)
By 藤枝　勇











![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/949.jpg)安藤玲二には一つの悩みがあった  

それは Featuer Match に全く勝てないことだった、 3 年前の日本選手権で行われた兄弟対決に勝って以来 1 勝しかしてないらしい。  

対するはデッキビルダーとしても名高い平林和哉 Pro Tour でも初日 6 - 0 - 1 という日本人最好成績もマークしている。  

はたして安藤はこの強敵を前に Feature Match に弱いというジンクスを破ることが出来るだろうか！？。


### Game 1


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/948.jpg)先手は安藤マリガン無し、後手は平林こちらもマリガン無し。  

先に動いたのは平林 2 ターン目に《 Werebear ／ 熊人間 》をだしマナ加速を開始、  

返しの安藤は《 Leaf Dancer ／ 葉の踊り手 》をキャストし、殴り合いに持っていく。  

平林 3 ターン目は《 Dreamwinder / 夢繰り 》安藤の場には島も有り、かなりのグッドスタッフである。  

安藤は 4 ターン目なにもせずにターンを返し平林は《 Dreamwinder / 夢繰り 》でアタック、安藤は手札に《 Elephant Ambush / 象の待ち伏せ 》があるがダメージレースに持っていくつもりかこれをスルー、エンドに《 Elephant Ambush / 象の待ち伏せ 》を打ち3 / 3 を戦線に投入し平林も同時に《 Elephant Ambush / 象の待ち伏せ 》をキャストしてにらみ合いになるかに見えたが、返しの安藤 5 ターン目《 Repel / 追い返し 》によって象トークンを除去した安藤はフルアタック、ターンを平林に返した。  

平林も負けじと夢繰りがアタックをして、また平林も《 Repel / 追い返し 》によって安藤の象トークンを排除する。  

ターンは進み安藤の《 Muscle Burst / 筋力急伸 》により《 Dreamwinder / 夢繰り 》が除去されるが平林は《 Aven Smokeweaver / エイヴンの煙の紡ぎ手 》を召喚し戦線を整える。  

安藤も《 Roar of the Wurm / ワームの咆哮 》トークンを場に出し一気に決めにかかろうとするが、平林の《 Kirtar's Desire / カーターの願望 》により阻止された上に《 Beast Attack / 獣の襲撃 》で 4 / 4 が出てしまい少し押され気味になるも2回目の《 Beast Attack / 獣の襲撃 》は《 Fervent Denial / 全面否定 》でカウンターしなんとか踏み留まる。  

安藤の《 Leaf Dancer / 葉の踊り手 》、平林の《 Aven Smokeweaver / エイヴンの煙の紡ぎ手 》は毎ターン、ダメージを与え続けライフが 安藤 4 対 平林 7 となっていた。  

そこで平林のメイン《 Sphere of Duty / 義務の領域 》が登場し安藤の攻撃が止まってしまい、平林の《 Aven Smokeweaver / エイヴンの煙の紡ぎ手 》が安藤のライフを削りきった。  

《 Aven Smokeweaver / エイヴンの煙の紡ぎ手 》には効かないが安藤が山を引き、手札にあった《 Thermal Blast / 炎熱の突風 》が使えればまた展開も変わったいただろう事を後述しておく。


平林 1 - 0 安藤


### Game 2


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/947.jpg)双方ともマリガンは無く順調な滑り出しが予想される。  

平林は Game 1 と変わらず 2 ターン目に《 Werebear / 熊人間 》をキャスト。  

先ほどと違うのは安藤が何もせずにターンを返したことか。  

平林は 3 ターン目にも《 Diligent Farmhand / 精励する農場労働者 》を出し順調なマナカーブを描く、対する安藤は四ターン目に 3 マナの《 Cephalid Looter / セファリッドの物あさり 》が出る今一ちぐはぐな展開。  

平林は 4 ターン目に《 Dreamwinder / 夢繰り 》を出し攻撃を開始、安藤も《 Krosan Archer / クローサの射手 》を出し、《 Dreamwinder / 夢繰り 》を受ける態勢となった。  

平林 6 ターン目の《 Dreamwinder / 夢繰り 》アタックを安藤は《 Cephalid Looter / セファリッドの物あさり 》と《 Krosan Archer / クローサの射手 》でダブルブロック、平林が両方に致死量を割り振ったところで安藤が《 Refresh / リフレッシュ 》をキャスト《 Krosan Archer / クローサの射手 》は Pump Up して《 Cephalid Looter / セファリッドの物あさり 》は再生、そのまま死んでいくかに見えた《 Dreamwinder / 夢繰り 》も平林が《 Refresh / リフレッシュ 》を打ち何も無かったかのようになる。  

返しのターンで安藤は《 Roar of the Wurm / ワームの咆哮 》をキャストするが平林はこれを《 Syncopate / 中略 》1度も出ないまま《 Roar of the Wurm / ワームの咆哮 》はリムーブされてしまった、安藤のエンドに平林は《 Diligent Farmhand / 精励する農場労働者 》をマナに変え8マナに到達し自分のターンに《 Rabid Elephant / 凶暴象 》《 Cephalid Looter / セファリッドの物あさり 》とキャストして一気に決めにかかる。安藤は《 Aven Windreader / エイヴンの風読み 》をブロッカーに廻すが平林の攻撃時に《 Dreamwinder / 夢繰り 》をブロックするが平林の《 Embolden / 励まし 》で相打ちがとれずきつい展開となってしまった。  

《 Kirtar's Desire / カーターの願望 》は、なんとか《 Fervent Denial / 全面否定 》でカウンターするもブロック毎にクリーチャーを減らし、そのまま負けてしまった。


平林 2 - 0 安藤


平林 Win
