Quarterfinal: 堂山 剛志 vs Olivier Ruel
===================================



 Posted in **Event Coverage**







![](https://media.magic.wizards.com/styles/auth_small/public/generic-avatar-150_144.png)
By 藤枝 勇











![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/884.jpg)2 人のドラフトは始まりこそ堂山《 Mystic Zealot / 秘教の盲信者 》Olivier が《 Scrivener / 公証人 》と共に WU を選択する流れだったが結果は対照的だった。  

上下共に RG を選択し WU の有効カードがおいしく流れてくる位置取りとなった堂山と違い Olivier は上下共に色が被ってしまった苦しい展開、結果しっかりとした WU が組めた堂山対してカードが足りずに 4 色というハンデ背負ってしまった Olivier 。  

ドラフトで開いた差を埋める事ができるだろうか？


### Game 1


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/883.jpg)堂山先手ランドはあるのだが島ばかり、手札は真っ白!苦い顔しながらもスタート。  

対する Olivier は順調な滑り出し 2 ターン目に《 Mystic Visionary / 秘教の幻想家 》を出す。  

堂山は 1 ターンだけだがランドが止まり先手後手が入れ替わってしまい苦しい展開となる。  

Olivier は 5 ターン目《 Rabid Elephant / 凶暴象 》をサモンし早くも攻撃態勢が整のった。  

堂山は Olivier の出した《 Cephalid Broker / セファリッドの仲介人 》に《 Repel / 追い返し 》を打ちドローを防ぎたかったのだが《 Shelter / 避難 》で回避され Olivier のデッキは軽快に回っていった。  

《 Rabid Elephant / 凶暴象 》に《 Kirtar's Desire / カーターの願望 》をつけ、何とか凌ごうとするが《 Scrivener / 公証人 》によって戻ってきた《 Shelter / 避難 》によって剥がされまたもや Olivier ペースで進んで行く。  

《 Beloved Chaplain / 敬愛される司祭 》《 Twigwalker / 小枝を歩くもの 》と防御も固めるが堂山の出した《 Aven Flock / エイヴンの群れ 》がどうにも止まらず少しずつライフが削られる Olivier 《 Rabid Elephant / 凶暴象 》の攻撃も《 Embolden / 励まし 》や《 Puppeteer / くぐつ師 》によってのらりくらりと回避していく堂山。  

《 Blessed Orator / 祝福された語り部 》が場に出、双方共に攻撃できない状況になると思われたが、 Olivier はメインから投入してある《 Execute / 処刑 》を打ち《 Aven Flock / エイヴンの群れ 》を除去するとスレッシュショルドが発動してる《 Mystic Visionary / 秘教の幻想家 》や《 Beloved Chaplain / 敬愛される司祭 》で徐々に押す展開となる。  

更に Olivier はここで駄目押しとなる《 Cultural Exchange / 文化交流 》を打ち堂山の《 Mystic Zealot / 秘教の盲信者 》《 Puppeteer / くぐつ師 》を奪うと、堂山は投了を宣言した。


Olivier 1 - 0 堂山


### Game 2


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/885.jpg)後手 Olivier はマリガンを宣言、やはり 4 色は辛いのか。  

堂山は《 Mystic Visionary / 秘教の幻想家 》を出し攻めたい所なのだが、《 Beloved Chaplain / 敬愛される司祭 》が出てしまい何もできなくなる。  

4 ターン目に《 Concentrate / 集中 》を打つなど堂山は手札に土地しかなく辛さが見え隠れする展開だ。  

ブロッカーとして出した《 Mystic Zealot / 秘教の盲信者 》も《 Zombie Assassin / ゾンビの暗殺者 》によって除去されてしまい、《 Beloved Chaplain / 敬愛される司祭 》を除去しようと《 Second Thoughts / 考え直し 》を打つも《 Rites of Refusal / 拒絶の儀式 》でカウンターされ苦しい展開となる。  

結局《 Beloved Chaplain / 敬愛される司祭 》は堂山のライフを 1 にするまでアタックを繰り返し、ここで引かなくては負けというところで堂山はなんとか《 Kirtar's Desire / カーターの願望 》を引き持ちこたえる。  

序盤ランドしか引かなかった堂山もここでようやく《 Patrol Hound / 巡視犬 》等を引いてくるが時既に遅し《 Cultural Exchange / 文化交流 》を打たれ勝負あり。


1 戦目 2 戦目共に土地事故という堂山には残念な結果となってしまった。


Olivier 2 - 0 堂山


Olivier Ruel Win
