1v1 Commander Changes
---------------------



It's time to turn up the power levels on 1v1 Commander! Check out [this article](https://magic.wizards.com/en/articles/archive/magic-online/magic-online-1v1-commander-update-2018-07-20) to see the changes that will be made. We think you'll be pretty excited. Also, on Saturday, we're having a 1v1 Commander Challenge!




---


Extra Sealed PTQ July 28
------------------------



Surprise! There will be an additional Pro Tour Qualifier event on July 28, at 7 a.m. PT. This event will feature *Core Set 2019* Sealed. To enter you must have a Chandra or Jace Limited PTQ Finals Entry Token. Note that we will not be having an event on Sunday, and that the schedule has been updated with PTQ's feeding PT Ravnica Allegiance.

Build Notes
-----------


Duel Scene | [Leovold, Emissary of Trest](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Leovold%2C+Emissary+of+Trest) now correctly prevents the optional card draw from [Sylvan Library](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Sylvan+Library).


Duel Scene | [Smallpox](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Smallpox) once again causes players to sacrifice lands and creatures.


Duel Scene | The game no longer resets when [Lazav, Dimir Mastermind](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Lazav%2C+Dimir+Mastermind) tries to copy a creature.


Collection | Premium full sets of Core Set 2019 can now be opened.


Collection | Several promo cards that were not properly displaying art are once again displaying art.


Collection | When Quantity is set to 0, unowned basic lands now appear.


Trade | Cards with meld are once again tradable.


1v1 Commander | The "partner" ability no longer functions in this format.


1v1 Commander | The banned cards list has been updated.
