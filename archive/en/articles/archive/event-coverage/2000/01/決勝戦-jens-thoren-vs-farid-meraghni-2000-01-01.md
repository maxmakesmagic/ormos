決勝戦: Jens Thoren vs Farid Meraghni
==================================



 Posted in **Event Coverage**







![](https://media.magic.wizards.com/styles/auth_small/public/generic-avatar-150_555.png)
By 石田格











![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/ptsd02/a466.jpg)プロツアーの決勝にしては珍しい事だがThorenとMeraghniはほぼ全くといっていいほど無名のプレイヤーである。
Thorenは多少 Grandprix 、欧州選手権等で戦績があるがMeraghniは正真正銘のルーキーで、どちらが勝っても新しい世代のプロツアーチャンピオンが誕生する事は間違いない。


### Game 1


開始早々サンディエゴ名物の《野生の雑種犬/[Wild Mongrel](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Wild+Mongrel)》を 2 ターン目に召喚するThorenに対しMeraghniは初めに登場するクリーチャーが《[Ember Beast](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Ember+Beast)/くすぶり獣 》という芳しくない立ち上がりである。


しかしThorenも続く《[Cartographer](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Cartographer)/ 地図作り 》を召喚した後はクリーチャーがぴたりと止まってしまう。その間Meraghniは《[Springing Tiger](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Springing+Tiger)/ 飛びかかる虎 》から《[Savage Firecat](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Savage+Firecat)/ 凶暴な火猫 》と完全にThorenをサイズで圧倒する。



全てを投げ打ってThorenは《野生の雑種犬/[Wild Mongrel](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Wild+Mongrel)》で《 Savage Firecat / 凶暴な火猫 》を相打ちにするが《[Savage Firecat](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Savage+Firecat)/ 凶暴な火猫 》が暴れている間には召喚できなかった《野生の雑種犬/[Wild Mongrel](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Wild+Mongrel)》や《[Nantuko Mentor](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Nantuko+Mentor)/ ナントゥーコの導師 》達が次々と場に現れ瞬く間にThorenは投了を余儀なくされた。


Meranghni 1-0


### Game 2


一本目を先制したMeraghniはその勢いに乗って《[Diligent Farmhand](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Diligent+Farmhand)/ 精励する農場労働者 》でのマナ加速から《[Springing Tiger](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Springing+Tiger) / 飛びかかる虎 》と後手の出遅れを完全に消し去る展開速度でThorenのライフを削りにかかる。


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/ptsd02/a465.jpg)Thorenも《葉の踊り手/[Leaf Dancer](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Leaf+Dancer)》でMeraghniのライフを削りながら《 [Mystic Zealot](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Mystic+Zealot) / 秘教の盲信者 》での防衛網を築きこうとする。しかしMeraghniはエンドステップにプレイした《[Elephant Ambush](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Elephant+Ambush) / 象の待ち伏せ 》のトークンと《 [Springing Tiger](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Springing+Tiger) / 飛びかかる虎 》を《無謀なる突進/[Reckless Charge](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Reckless+Charge)》で強化してThorenのライフを一気に削りとろうとする。


数ターン後度重なるMeraghniの猛攻を《 [Angelic Wall](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Angelic+Wall) / 天使の壁 》や《 [Mystic Zealot](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Mystic+Zealot) / 秘教の盲信者 》を使いつぶしつつどうにか凌いだThorenはようやく《獣の襲撃/[Beast Attack](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Beast+Attack)》から一気に逆襲を開始する。


Meraghniも手札にある《[Kamahl, Pit Fighter](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Kamahl%2C+Pit+Fighter) / ピット・ファイター、カマール 》を出せればThorenのライフを削りきれるのだが6マナは果てしなく遠く、無敵の戦士がMeraghniを救う前に《葉の踊り手/[Leaf Dancer](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Leaf+Dancer)》がMeraghniのライフを削りきった。


Thoren 1-1


### Game 3


相変わらず勢いの衰えないMeraghniは《野生の雑種犬/Wild Mongrel》、《 [Werebear](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Werebear) / 熊人間 》、《 [Cephalid Looter](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Cephalid+Looter) / セファリッドの物あさり 》、《 [Savage Firecat](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Savage+Firecat) / 凶暴な火猫 》とスピード、サイズ共にThorenを完全に圧倒する。
Thorenも《獣の襲撃/[Beast Attack](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Beast+Attack)》のトークンと《凶暴象/[Rabid Elephant](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Rabid+Elephant)》でのダブルブロックで《 [Savage Firecat](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Savage+Firecat) / 凶暴な火猫 》を除去するが《炎の稲妻/[Firebolt](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Firebolt)》で生き残った《凶暴象/Rabid Elephant》も《葉の踊り手/[Leaf Dancer](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Leaf+Dancer)》ごと除去されてしまう。


さらに《セファリッドの物あさり/[Cephalid Looter](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Cephalid+Looter)》の効果によりスレッショルドした《[Springing Tiger](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Springing+Tiger)/ 飛びかかる虎 》と《[Werebear](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Werebear)/ 熊人間 》に押される状況を《ゴリラのタイタン/[Gorilla Titan](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Gorilla+Titan)》や《[Beast Attack](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Beast+Attack)/ 獣の襲撃 》のトークンでなんとか持ちこたえさせるThorenだったがMeraghniの《大洪水/[Deluge](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Deluge)》の前についに力尽きた。


Meraghni 2-1


### Game 4


もはや後が無いThorenは鬼気迫る形相で《葉の踊り手/[Leaf Dancer](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Leaf+Dancer)》を召喚してターンクロックをかけつつ《獣の襲撃/[Beast Attack](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Beast+Attack)》のトークンや《遊牧の民のおとり/[Nomad Decoy](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Nomad+Decoy)》を駆使してMeraghniの《呪文散らしのケンタウルス/[Spellbane Centaur](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Spellbane+Centaur)》や《 [Springing Tiger](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Springing+Tiger) / 飛びかかる虎 》、さらには《 [Nantuko Disciple](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Nantuko+Disciple) / ナントゥーコの信奉者》までも完全に封殺することに成功する。


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/ptsd02/a467.jpg)勝負どころで冴えないドローを繰り返すMeraghniを尻目にThorenは《石舌のバジリスク/[Stone-Tongue Basilisk](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Stone-Tongue+Basilisk)》と《[Thaumatog](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Thaumatog)/ ソーマトグ》を場に出し隙あらばいつでもMeraghniを倒せる状態にして油断なく《葉の踊り手/[Leaf Dancer](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Leaf+Dancer)》で攻め続ける、《大洪水/[Deluge](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Deluge)》さえ引ければMeraghniにもまだ勝機があるのだがMeraghniが土地以外のカードを引くことは無かった…。


Thoren 2-2


### Game 5


泣いても笑ってもこれが最後の最後、勝った方が笑い負けた方が涙する。


しかし、ここでついにMeraghniは《 [Werebear](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Werebear) / 熊人間 》と《 [Nantuko Elder](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Nantuko+Elder) / ナントゥーコの古老 》から《 [Kamahl, Pit Fighter](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Kamahl%2C+Pit+Fighter) / ピット・ファイター、カマール 》を召喚し脅威の固定砲台を場に設置することに成功する。


戦前から言われていた事だがThorenのデッキは《 [Kamahl, Pit Fighter](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Kamahl%2C+Pit+Fighter) / ピット・ファイター、カマール 》に対する耐性は無いに等しい。完全に無敵の戦士と化した《 [Kamahl, Pit Fighter](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Kamahl%2C+Pit+Fighter) / ピット・ファイター、カマール 》はThorenのクリーチャーを端から真っ二つに切り刻んでいく。


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/ptsd02/a461.jpg)完全に進退窮まったThorenはクリーチャーが全滅する前に《踏み荒らし/Overrun》＋《 [Thaumatog](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Thaumatog) / ソーマトグ 》で最期の攻勢に出る。


しかし、土地を生贄に捧げた時点でMeraghniは手札から《 [Moment's Peace](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=Moment%27s+Peace) / 一瞬の平和 》を見せこの戦いに終止符を打ったのだった。


Final Result : Meraghni　Win　3-2  

Congratulation Meraghni!!
