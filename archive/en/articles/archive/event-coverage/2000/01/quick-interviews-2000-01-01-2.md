Quick Interviews
================



 Posted in **Event Coverage**







![](https://media.magic.wizards.com/styles/auth_small/public/images/person/authorpic_KeitaMori.jpg)
By Keita Mori











**今大会最もキツかった対戦は ?**




|  |  |  |
| --- | --- | --- |
|  |  |  |
| **Michael Long:** 初日最終戦、初日全勝したヤツだろうな。でもまあ、アラホリの優勝で決まりだろ（Long は決勝を見届けずに撤収） | **荒堀和明:** 平林さん。さすがに最終戦ダブルマリガンの瞬間は覚悟しました。 | **平林和哉:** 荒堀君。予選決勝とも勝てなかったなあ。 |

**今日のあなたの勝利を支えててくれた要因は？**




|  |  |  |
| --- | --- | --- |
|  |  |  |
| **荒堀和明(初日終了時点 3 位):** 《ゾンビの横行/Zombie Infestation》ですね。 | **Tomoya Tsushima（初日唯一の全勝）:** 《怨恨/Rancor》でした。 | **小野田倫久（初日終了時点 2 位）:** 《剣を鍬に/Swords to Plowshares》かな。 |

|  |  |  |
| --- | --- | --- |
|  |  |  |
