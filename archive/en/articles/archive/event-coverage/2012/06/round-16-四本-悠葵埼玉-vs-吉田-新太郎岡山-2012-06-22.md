Round 16: 四本 悠葵(埼玉) vs. 吉田 新太郎(岡山)
==================================



 Posted in **Event Coverage**
 on June 22, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_author.jpg)
By Wizards of the Coast











　長かったグランプリ横浜もいよいよ最終ラウンド。すでに多くのトップ8が決まる中、最後の望みをかけて2人のプレイヤーが席についた。 


　両プレイヤーの成績は37ポイント、12勝2敗1分。この試合を勝利すれば39ポイント、13勝2敗で握手をしているプレイヤーに追いつくことができる。 


　1人は0BYEからの初日無敗という成績を残した四本 悠葵(埼玉)。使うデッキは赤緑白の《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》、彼もまた先のカバレッジで紹介された遠藤 亮太(埼玉)のデッキを使う1人である。同じデッキを使っている角岡 利幸(東京)はすでにトップ8を決めているが共にトップ8に残ることができるのか。 


　対するは岡山の吉田 新太郎、何の因果か彼がこのグランプリの2日目、唯一黒星をつけた相手が使っていたデッキもまたこの《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》であった。この試合はそのリベンジマッチでもある。 


　そんな吉田が使用するデッキは無色トロン。最速3ターン目に降臨する《[解放された者、カーン](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A7%A3%E6%94%BE%E3%81%95%E3%82%8C%E3%81%9F%E8%80%85%E3%80%81%E3%82%AB%E3%83%BC%E3%83%B3)》は数多くのデッキを脅かしてきたが、この試合ではその姿を見ることはできるだろうか。 


　泣いても笑っても最後の一戦、果たしてトップ8への希望をつなぐことができるのはどちらか！ 


![R16_yoshida_VS_yotsumoto.jpg](https://media.magic.wizards.com/image_legacy_migration/images/magic/daily/eventsjp/gpyok12/R16_yoshida_VS_yotsumoto.jpg)


### ゲーム１


　先攻は四本、初期手札の7枚が良いカードであるよう手を合わせる。その祈りが通じたのか7枚を見てすぐさまキープを宣言。一方後攻の吉田はダブルマリガン。「前の試合もそうだったんだよ」と苦笑い。 


　四本は1ターン目に《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》、2ターン目に《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》と最高の滑り出し。しかし5枚でスタートした吉田も《[ウルザの鉱山](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83314)》《[ウルザの魔力炉](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83315)》から《[探検の地図](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%8E%A2%E6%A4%9C%E3%81%AE%E5%9C%B0%E5%9B%B3)》と動きでは負けていない。 


　3ターン目、四本は土地をセットするのみでターンを返すと、吉田はなんと手札から《[ウルザの塔](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83316)》。ライブラリーから探すことなくウルザ地形を3種揃えたばかりか、さらに飛び出してくる最速の《[解放された者、カーン](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A7%A3%E6%94%BE%E3%81%95%E3%82%8C%E3%81%9F%E8%80%85%E3%80%81%E3%82%AB%E3%83%BC%E3%83%B3)》！　[-3]で四本の《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》を追放する。 


　だがキーカードを追放されてしまったはずの四本は逆に笑みを浮かべる。吉田のターンエンドに《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》が、そして自分のターンに入って《[鏡割りのキキジキ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%8F%A1%E5%89%B2%E3%82%8A%E3%81%AE%E3%82%AD%E3%82%AD%E3%82%B8%E3%82%AD)》が。一瞬にして無限トークンコンボが決まった。 


　わずか4ターン、超高速の戦いは四本が先制！ 


*四本 1-0 吉田*



### ゲーム２




|  |
| --- |
|  |
| *吉田 新太郎* |

　先程は5枚でスタートだった吉田、今回は7枚からのスタート。四本も初期手札をキープする。 　まずはお互い《[彩色の宝球](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=46046)》、《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》と先ほどと全く同じスタートを切る。 


　吉田は2ターン目、先ほどの《[の宝球](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=46046)》から緑マナを出し《[森の占術](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A3%AE%E3%81%AE%E5%8D%A0%E8%A1%93)》。すでに《[ウルザの塔](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83316)》《[ウルザの鉱山](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83314)》を置いているので探してくるのはもちろん《[ウルザの魔力炉](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83315)》。 


　そう簡単に揃わせるわけにはいかない、四本は《[大爆発の魔道士](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A4%A7%E7%88%86%E7%99%BA%E3%81%AE%E9%AD%94%E9%81%93%E5%A3%AB)》を唱え、すぐさま《[ウルザの塔](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83316)》を破壊。これにより吉田は《[ウルザの魔力炉](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83315)》をセットするしかできず3ターン目を終える。 


　しかし吉田の手札には《[ウルザの塔](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83316)》、わずか1ターンで揃い直されたウルザ地形から今回も《[解放された者、カーン](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A7%A3%E6%94%BE%E3%81%95%E3%82%8C%E3%81%9F%E8%80%85%E3%80%81%E3%82%AB%E3%83%BC%E3%83%B3)》が出る。 　前ターンに四本の置いた《[呪文滑り](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%91%AA%E6%96%87%E6%BB%91%E3%82%8A)》を避けて[+4]能力を使用。四本の手札から《[召喚の調べ](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89064)》を追放した。 


　大量にマナを使用できる吉田に対して、四本は3枚目の土地を引けず《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》を置くだけでターンエンド、 


　吉田の5ターン目、《[解放された者、カーン](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A7%A3%E6%94%BE%E3%81%95%E3%82%8C%E3%81%9F%E8%80%85%E3%80%81%E3%82%AB%E3%83%BC%E3%83%B3)》がもう一枚《[召喚の調べ](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=89064)》を追放し、《[彩色の星](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BD%A9%E8%89%B2%E3%81%AE%E6%98%9F)》から出した赤マナで《[紅蓮地獄](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%B4%85%E8%93%AE%E5%9C%B0%E7%8D%84)》で土地が止まっている四本の《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》を落とす。さらに《[世界のるつぼ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%B8%96%E7%95%8C%E3%81%AE%E3%82%8B%E3%81%A4%E3%81%BC)》によりこれ以上の土地破壊が無意味なことを示され四本の反撃の目は潰えた。 


*四本 1-1 吉田*



### ゲーム３




|  |
| --- |
|  |
| *四本 悠葵* |

　まだこの試合が始まって9ターンしか経過していない。超高速でコンボを決める2人の最終ゲーム。先攻は四本。 


　四本は余裕を持ってキープを宣言。吉田は少しばかり悩むが意を決したように7枚の手札をキープした。 


　このゲームもまた1ターン目に《[極楽鳥](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E6%A5%BD%E9%B3%A5)》を出した四本、2ターン目の《[大爆発の魔道士](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%A4%A7%E7%88%86%E7%99%BA%E3%81%AE%E9%AD%94%E9%81%93%E5%A3%AB)》が吉田の《[ウルザの魔力炉](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83315)》を破壊する。 


　すると吉田は2ターン目、土地をセットできずそのまま四本へとターンを返すことを余儀なくされる。 


　3ターン目、四本が土地を置いただけでエンドとデジャビュを感じる展開に、かろうじて《[ウルザの鉱山](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=83314)》を引いた吉田は、1ターン目に出した《[彩色の宝球](https://gatherer.wizards.com/Pages/Card/Details.aspx?multiverseid=46046)》による緑マナから《[古きものの活性](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%8F%A4%E3%81%8D%E3%82%82%E3%81%AE%E3%81%AE%E6%B4%BB%E6%80%A7)》を唱える。しかしそこに土地はなくしぶしぶ《[探検の地図](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%8E%A2%E6%A4%9C%E3%81%AE%E5%9C%B0%E5%9B%B3)》を手札に加える。 


　ターンエンドを宣言すると、そこには笑みを隠し切れない四本。ターンエンドに出る《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》、そして《[鏡割りのキキジキ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%8F%A1%E5%89%B2%E3%82%8A%E3%81%AE%E3%82%AD%E3%82%AD%E3%82%B8%E3%82%AD)》。最初のゲームと同じ4ターン目、最速のコンボ達成で四本がトップ8の切符を掴みとった。 


*四本 2-1 吉田*
