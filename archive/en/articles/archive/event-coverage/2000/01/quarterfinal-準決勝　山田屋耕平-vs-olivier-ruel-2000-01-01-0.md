Quarterfinal: 準決勝　山田屋耕平 vs. Olivier Ruel
========================================



 Posted in **Event Coverage**







![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_author.jpg)
By Wizards of the Coast











![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/868.jpg)これに勝てばいよいよ決勝、気合の入る対戦である。両者共に緑主体の多色デッキであり、山田屋のとんでもないクリーチャー陣を Olivier が《 Cultural Exchange / 文化交流 》で奪い取れるか、そこに注目したいところだ｡


### Game 1


先手の山田屋は土地 5 枚に《 Chatter of the Squirrel / リスのお喋り 》、《 Shelter / 避難 》と言うまずまずの手札でスタート。一方 Olivier はマリガン後に《 Diligent Farmhand / 精励する農場労働者 》とともに《 Zombie Assassin / ゾンビの暗殺者 》などのクリーチャーをかかえてひとまず安心のスタートとなった｡  

リストークンは《 Diligent Farmhand / 精励する農場労働者 》と相打ちになり、序盤のクリーチャーはひとまず全滅、中盤の中～大型クリーチャーが現れ始める｡  
![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/869.jpg)

山田屋が《 Ivy Elemental / キヅタの精霊 》を 4 / 4 で召喚、これを Olivier は《 Krosan Archer / クローサの射手 》と《 Zombie Assassin / ゾンビの暗殺者 》でダブルブロック｡《 Second Thoughts / 考え直し 》でうまく山田屋の《 Shelter / 避難 》を誘い出してからゾンビで葬った｡そのあと山田屋の場には《 Cephalid Scout / セファリッドの斥候 》、《 Shelter / 避難 》を拾った《 Scrivener / 公証人 》が、 Olivier の場には《 Rabid Elephant / 凶暴象 》と《 Krosan Archer / クローサの射手 》が並ぶことになる｡ここへ飛び込もうとした《 Aboshan， Cephalid Emperor / セファリッドの皇帝アボシャン 》は待ち構えていたように《 Syncopate / 中略 》でカウンターされ、長期戦になることが予想された。  

ランドを次々とつぶしてドローを続けた山田屋は待望の《 Iridescent Angel / 玉虫色の天使 》を手に入れ、これを場に出すことに成功した｡ Olivier にとっては、というよりもあらゆるデッキにとってとんでもない脅威であるこの天使は着実にダメージを与えつづける｡ Olivier にとって最後の望みは《 Cultural Exchange / 文化交流 》だけであるが、残念ながら最後までライブラリーのトップに来ることは無かった｡


山田屋 1 - 0 Olivier


![](https://media.magic.wizards.com/image_legacy_migration/sideboard/images/gpshi01/870.jpg)### Game 2


山田屋は土地 2 枚に《 Diligent Farmhand / 精励する農場労働者 》を初手に発見し、安心したスタートを切る。一方 Olivier はまたしてもマリガンし、少々いらついたのか土地を置き間違えたせいで 2 ターン目に《 Mystic Visionary / 秘教の幻想家 》を召喚できないというプレイミス｡それでも何とか 4 ターン目の《 Nantuko Disciple / ナントゥーコの信奉者 》までつなげ、山田屋の攻撃を食い止めた｡  

中盤戦、山田屋は《 Zombie Assassin / ゾンビの暗殺者 》を《 Syncopate / 中略 》でカウンターし、すぐに《 Scrivener / 公証人 》で拾い上げる｡ Olivier が《 Beloved Chaplain / 敬愛される司祭 》を召喚し、場は完全に膠着しつつあった｡ただ、 Olivier の《 Scrivener / 公証人 》がカウンターされ、墓地の《 Syncopate / 中略 》が拾えなくなったので、山田屋が天使を引いてしまえばそれで勝負がつくことが明白になった｡この環境ではほとんど見ない 4 色デッキの弊害か、 Olivier はいつまでたっても森を引けずに手札の緑クリーチャーも腐っており、かなり苦しい事態である｡《 Cephalid Looter / セファリッドの物あさり 》を召喚した山田屋は、とうとう《 Aboshan, Cephalid Emperor / セファリッドの皇帝アボシャン 》を召喚、先に出していた《 Stone-Tongue Basilisk / 石舌のバジリスク 》もあわさって、 Olivier はまたしても《 Cultural Exchange / 文化交流 》頼みになってしまった｡しかし今回もその望みは適わず、最後に場違いな《 Cease-Fire / 消えた火 》を打ち上げてゲームが終了となった｡


山田屋 2 - 0 Olivier  

山田屋 Wins
