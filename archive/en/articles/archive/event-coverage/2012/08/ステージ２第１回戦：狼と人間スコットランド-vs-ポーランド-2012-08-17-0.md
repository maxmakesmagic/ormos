ステージ２第１回戦：狼と人間スコットランド vs. ポーランド
===============================



 Posted in **Event Coverage**
 on August 17, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/generic-avatar-150_56.png)
By Tim Willougby











　モダン卓で、スコットランドの国内チャンピオン、アンドリュー・モリソン/Andrew Morrisonがポーランドの国内チャンピオン、マテウス・コペック/Mateusz Kopecと向かい合った。だが、彼らの試合が２国間の戦いの口火を切ったわけではなかった。




|  |
| --- |
|  |
| *ポーランドとスコットランドの戦いには国のプライド以上のものが賭かっている* |

  
　両者マリガンによって、ブロック構築が最初にキック・オフとなる。ジャンドの同型戦のようで、マナ・クリーチャーで初期の体勢を整えたのち、ポーランドが《[情け知らずのガラク](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%83%85%E3%81%91%E7%9F%A5%E3%82%89%E3%81%9A%E3%81%AE%E3%82%AC%E3%83%A9%E3%82%AF)》で狼を生み出し、機先を制した。


　ガラクによる盤面の優位は長くは続かなかった。《[ファルケンラスの貴種](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A1%E3%83%AB%E3%82%B1%E3%83%B3%E3%83%A9%E3%82%B9%E3%81%AE%E8%B2%B4%E7%A8%AE)》がプレインズウォーカーを退場させた。《[ファルケンラスの貴種](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A1%E3%83%AB%E3%82%B1%E3%83%B3%E3%83%A9%E3%82%B9%E3%81%AE%E8%B2%B4%E7%A8%AE)》を倒すのに２枚目の《[情け知らずのガラク](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%83%85%E3%81%91%E7%9F%A5%E3%82%89%E3%81%9A%E3%81%AE%E3%82%AC%E3%83%A9%E3%82%AF)》を切ったが、この交換はスコットランドのブラッドリー・バークレイ/ Bradley Barclayには良いものだったようだ。


　次のバークレイからの積極的な動きは《[高原の狩りの達人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AB%98%E5%8E%9F%E3%81%AE%E7%8B%A9%E3%82%8A%E3%81%AE%E9%81%94%E4%BA%BA)》だったが、相手の《[忌むべき者のかがり火](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BF%8C%E3%82%80%E3%81%B9%E3%81%8D%E8%80%85%E3%81%AE%E3%81%8B%E3%81%8C%E3%82%8A%E7%81%AB)》が立ちはだかり、維持することは叶わなかった。かがり火がバークレイを強烈なパンチが支配する試合へと引き戻した。


　無人となった盤面に、ポーランドのトメク・ペドラコウスキー/ Tomek Pedrakowskiがもうひとつ強烈なパンチを放った。初めに《[ウルフィーの銀心](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A6%E3%83%AB%E3%83%95%E3%82%A3%E3%83%BC%E3%81%AE%E9%8A%80%E5%BF%83)》、それから《[アヴァシンの巡礼者](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A2%E3%83%B4%E3%82%A1%E3%82%B7%E3%83%B3%E3%81%AE%E5%B7%A1%E7%A4%BC%E8%80%85)》で結魂。バークレイはこのワン・ツーの片方にしか対抗できず、ポーランドの攻撃では、狼１体にチャンプ・ブロックを何度か繰り返した。チャンプ・ブロックを交換へと変えたのは《[悲劇的な過ち](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%82%B2%E5%8A%87%E7%9A%84%E3%81%AA%E9%81%8E%E3%81%A1)》で、さらにフラッシュバックした《[小悪魔の遊び](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B0%8F%E6%82%AA%E9%AD%94%E3%81%AE%E9%81%8A%E3%81%B3)》が盤面を片付けた。なんとか均衡を得ようとする努力は及ばず、バークレイは続くターンの《[ファルケンラスの貴種](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A1%E3%83%AB%E3%82%B1%E3%83%B3%E3%83%A9%E3%82%B9%E3%81%AE%E8%B2%B4%E7%A8%AE)》による素早い攻撃に圧倒され続けた。


　ちょうどバークレイが第１ゲームを落とした頃、仲間のスティーブン・マーレイ/ Stephen Murrayが青白デルバーでアダム・ビューバックズ/Adam Bubacz操るゾンビ・デッキに最後まで力強く勝利を収めていた。複数の《[はらわた撃ち](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%81%AF%E3%82%89%E3%82%8F%E3%81%9F%E6%92%83%E3%81%A1)》が最初の猛攻を止める助けになり、止まる頃には《[聖トラフトの霊](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%81%96%E3%83%88%E3%83%A9%E3%83%95%E3%83%88%E3%81%AE%E9%9C%8A)》と《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》がゲームを片付けた。


　真ん中の卓では、アンドリュー・モリソンがマテウス・コペックとの戦いで身動きが封じられていた。コペックの青赤緑ネクスト・レベル・ブルーが、ジャンドの攻撃を遅らせるのに長けている、ということを証明しているのだ。こちらのポーランド人はグランプリ・ウィーンを同様のデッキで勝ち取っていて、《[ヴィダルケンの枷](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%B4%E3%82%A3%E3%83%80%E3%83%AB%E3%82%B1%E3%83%B3%E3%81%AE%E6%9E%B7)》を使って攻撃にもうひとつ要素を加えることで相手を混乱させることにかけては、絶対の自信を持っているようだった。コペックが完全に場の支配を得るまでの間、《[ヴィダルケンの枷](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%B4%E3%82%A3%E3%83%80%E3%83%AB%E3%82%B1%E3%83%B3%E3%81%AE%E6%9E%B7)》は十全に事を成した。それはつまり、ポーランドが第２ゲームへと進むのに少しだけ有利だった、ということだ。




|  |
| --- |
|  |
| *ポーランドの対戦相手に向かい合い、誇りを持って国旗をまとうスコットランド・チーム* |

  
　ブロック構築では、ペドラコウスキーがマリガンをし、２ターン目に出たバークレイの《[軽蔑された村人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%BB%BD%E8%94%91%E3%81%95%E3%82%8C%E3%81%9F%E6%9D%91%E4%BA%BA)》が裏返るのをただ見ていることしかできず、最速《[ウルフィーの銀心](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A6%E3%83%AB%E3%83%95%E3%82%A3%E3%83%BC%E3%81%AE%E9%8A%80%E5%BF%83)》を許してしまった。バークレイには赤マナがなく、立ち往生する可能性があったが、彼はそれを回避する良い場所にいたようだ。ペドラコウスキーの《[オリヴィア・ヴォルダーレン](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AA%E3%83%AA%E3%83%B4%E3%82%A3%E3%82%A2%E3%83%BB%E3%83%B4%E3%82%A9%E3%83%AB%E3%83%80%E3%83%BC%E3%83%AC%E3%83%B3)》でも相手の速さを落とすことはなかった。《[血統の切断](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A1%80%E7%B5%B1%E3%81%AE%E5%88%87%E6%96%AD)》が道を開け、ブロック構築を1-1に持ち込んだ。


　モダンでは、コペックがギアを切り替えたようだ。第２ゲームは《[高原の狩りの達人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AB%98%E5%8E%9F%E3%81%AE%E7%8B%A9%E3%82%8A%E3%81%AE%E9%81%94%E4%BA%BA)》でスコットランド側に攻め込み、よりアグレッシブなものとなっている。この好戦的なゲームはブロック構築さながらで、ジャンド側は毎ターン脅威に除去を費やし、コペックを叩いていた。


　しかしどういうわけか、もっともアグレッシブなのは青白デルバーだった。《[聖トラフトの霊](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%81%96%E3%83%88%E3%83%A9%E3%83%95%E3%83%88%E3%81%AE%E9%9C%8A)》と《[ルーン唱えの長槍](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%AB%E3%83%BC%E3%83%B3%E5%94%B1%E3%81%88%E3%81%AE%E9%95%B7%E6%A7%8D)》、そして複数の《[蒸気の絡みつき](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%92%B8%E6%B0%97%E3%81%AE%E7%B5%A1%E3%81%BF%E3%81%A4%E3%81%8D)》がアダム・ビューバックズのゾンビを押さえ込み、打ちのめし、スコットランドのスティーブン・マーレイが最初に勝利をものにした。


スコットランド 1-0 ポーランド




|  |
| --- |
|  |
| *１勝目を取ったのはチーム・スコットランド。* |

  
　ブロック構築では、面白い流れになっていた。バークレイの《[高原の狩りの達人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AB%98%E5%8E%9F%E3%81%AE%E7%8B%A9%E3%82%8A%E3%81%AE%E9%81%94%E4%BA%BA)》が、相手の場の《[魔女封じの宝珠](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AD%94%E5%A5%B3%E5%B0%81%E3%81%98%E3%81%AE%E5%AE%9D%E7%8F%A0)》で《[高原の荒廃者](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AB%98%E5%8E%9F%E3%81%AE%E8%8D%92%E5%BB%83%E8%80%85)》に変身する際の能力を削がれ、少しばかり強さを減らされていたのだ。ペドラコウスキーの方の《[高原の狩りの達人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AB%98%E5%8E%9F%E3%81%AE%E7%8B%A9%E3%82%8A%E3%81%AE%E9%81%94%E4%BA%BA)》は力を削がれていないもので、ポーランド側に残っている《[高原の狩りの達人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AB%98%E5%8E%9F%E3%81%AE%E7%8B%A9%E3%82%8A%E3%81%AE%E9%81%94%E4%BA%BA)》を《[血統の切断](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A1%80%E7%B5%B1%E3%81%AE%E5%88%87%E6%96%AD)》で巻き込む前にバークレイの攻撃を誘って、彼の盤面をあらかた処分した。


　ペドラコウスキーは、《[ヴェールの呪いのガラク](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%B4%E3%82%A7%E3%83%BC%E3%83%AB%E3%81%AE%E5%91%AA%E3%81%84%E3%81%AE%E3%82%AC%E3%83%A9%E3%82%AF)》の−１能力で狼男たちを追加し、バークレイを危険地帯へと追い込んだ。その間に、コペックがモダン戦を勝ち取り、ブロック構築のマッチをこの試合のタイ・ブレークに位置づけた。


スコットランド 1-1 ポーランド




|  |
| --- |
|  |
| *チーム・ポーランドがスコットランドに追いつき、すべてのプレッシャーが３戦目にのしかかる。* |

  
　バークレイは《[国境地帯のレインジャー](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%9B%BD%E5%A2%83%E5%9C%B0%E5%B8%AF%E3%81%AE%E3%83%AC%E3%82%A4%E3%83%B3%E3%82%B8%E3%83%A3%E3%83%BC)》と《[ファルケンラスの貴種](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A1%E3%83%AB%E3%82%B1%E3%83%B3%E3%83%A9%E3%82%B9%E3%81%AE%E8%B2%B4%E7%A8%AE)》では後者がガラクに対する最高の回答であると判断した。ペドラコウスキーは嬉しそうに、さらに大きな脅威を戦場に投下したが、しかし、バークレイのライフ総量からすれば目の前にはっきりと存在する脅威は《[ウルフィーの銀心](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A6%E3%83%AB%E3%83%95%E3%82%A3%E3%83%BC%E3%81%AE%E9%8A%80%E5%BF%83)》だった。ライフは攻撃後10点となり、バークレイは彼が減らさなくてはならない24点を見据え、《[ファルケンラスの貴種](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A1%E3%83%AB%E3%82%B1%E3%83%B3%E3%83%A9%E3%82%B9%E3%81%AE%E8%B2%B4%E7%A8%AE)》をブロックに回すより攻撃させることを選んだ。《[血統の切断](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E8%A1%80%E7%B5%B1%E3%81%AE%E5%88%87%E6%96%AD)》のフラッシュバックで《[ウルフィーの銀心](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A6%E3%83%AB%E3%83%95%E3%82%A3%E3%83%BC%E3%81%AE%E9%8A%80%E5%BF%83)》を対処するが、ポーランド側にはすぐさま２体目が置かれた。


　３体目の《[ウルフィーの銀心](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A6%E3%83%AB%E3%83%95%E3%82%A3%E3%83%BC%E3%81%AE%E9%8A%80%E5%BF%83)》がペドラコウスキーから繰り出され、彼はレッド・ゾーンで膨大な量のダメージを見せつけた。バークレイは《[悲劇的な過ち](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%82%B2%E5%8A%87%E7%9A%84%E3%81%AA%E9%81%8E%E3%81%A1)》のコンバット・トリックで何体かを止められたが、すべてを止めることはできず、すぐ次のターン、彼はカードをまとめた。
