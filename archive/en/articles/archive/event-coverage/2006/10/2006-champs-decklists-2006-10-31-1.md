2006 Champs Decklists
=====================



 Posted in **Event Coverage**
 on October 31, 2006 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_author.jpg)
By Wizards of the Coast











The 2006 **Magic** State/Province/Territory/Island Champs assembled thousands of **Magic** players across dozens of states, provinces, territories, islands, cities, counties and prefectures, providing the first chance to play *Time Spiral* cards in a Standard event.


As an added bonus, all the decklists are provided as ***Magic**  Online*  deck files in text (.txt) format. To download a deck, right-click the ".txt" icon in the upper-right corner of that deck and select "Save As..." Save that text file in your Program Files\Magic Online\decks folder. Then run ***Magic**  Online*  and load the text file to see the deck. For more information about using text files with ***Magic**  Online*, [click here](http://archive.wizards.com/Magic/Magazine/Article.aspx?x=mtgcom/arcana/678).


North American decklists and winners' photos can be found [here](http://archive.wizards.com/Magic/Magazine/Article.aspx?x=mtgevent/champsmap06/standard).
