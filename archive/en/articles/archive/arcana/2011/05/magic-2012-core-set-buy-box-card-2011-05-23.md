***Magic** 2012* Core Set Buy-a-Box Card
========================================



 Posted in **Arcana**
 on May 23, 2011 






![](https://web.archive.org/web/20211024110319im_/https://magic.wizards.com/sites/all/themes/wiz_mtg/images/global/generic-avatar-150.png)
By Monty Ashley











You might have heard of the "buy-a-box" card that we mention each time there's a new set. It's a free alternate-art premium card that's given to the first twenty people to buy a Booster Box at participating locations. And if you hadn't, you have now! The Buy-a-Box card for the ***Magic** 2012* Core Set is Chandra's Phoenix, and it looks like this!


![Chandra's Phoenix](https://media.magic.wizards.com/image_legacy_migration/images/magic/daily/arcana/716_5es3uztmmk.jpg)  
Promotion mechanic might vary by region.
