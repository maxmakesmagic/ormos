第４回戦：アメリカ対チーム・アメリカBrian Kibler(アメリカ) vs. Daniel Antoniou(キプロス)
==============================================================



 Posted in **Event Coverage**
 on August 17, 2012 






![](https://web.archive.org/web/20211024110319im_/https://magic.wizards.com/sites/all/themes/wiz_mtg/images/global/generic-avatar-150.png)
By Tim Willoughby











### Brian Kibler(ナヤ《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》) vs. Daniel Antoniou(赤白青コントロール)


　ブライアン・キブラー/Brian Kiblerは、国内王者のメダルを引っさげて、フィーチャーマッチ・エリアでくつろぎ、シャッフルをしてマッチの開始を待っていた。《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》を中心にした、ナヤ（緑赤白）デッキをプレイしていると――後付けかもしれないが、彼はそうしたデッキでこれまでに成し遂げた業績をなぞろうとしているのではないか、と思われた。プロツアー「闇の隆盛」で優勝した「Zoo」系デッキ然り、あるいは彼がドラゴンマスターの名を冠することになったプロツアー・シカゴ2000のデッキ然り。シャッフルをしている間、キブラーはこのトーナメントのために選んだ曲、Tritonal（訳注：アメリカのトランス・ミュージシャン）の『Can't Keep It In』を聴いて自分を高めていた。彼にはその３色がよく似合う。


　キブラーの対戦相手、キプロスのダニエル・アントニーオ/Daniel Antoniouは、経験という点ではわずかに遅れをとっていた。キプロスはこれまで一度も世界選手権に代表チームを送り込んでいなかったが、その島のチャンピオンとして、彼はそのチームのメダルを提げていた。アントニーオの最初のプロツアーはプロツアー「闇の隆盛」だったが、彼のチームの世界での存在感は小さかったため、それでチャンピオンの座につくに十分だった。また皮肉なことに、アントニーオは赤白青コントロール――チーム・アメリカと称されるデッキで戦うことを選んでいたのだった。

**ゲーム１**


　プレイに入ると、キブラーはマリガンを余儀なくされた。しかしチームUSAのキャプテンは、《[魂の洞窟](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AD%82%E3%81%AE%E6%B4%9E%E7%AA%9F)》を使って打ち消し呪文を避けながらの第２ターン《[スレイベンの守護者、サリア](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B9%E3%83%AC%E3%82%A4%E3%83%99%E3%83%B3%E3%81%AE%E5%AE%88%E8%AD%B7%E8%80%85%E3%80%81%E3%82%B5%E3%83%AA%E3%82%A2)》を出してみせた。アントニーオの初動は《[思案](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%80%9D%E6%A1%88)》で、そのデッキはチームUSAをあざ笑うかのように淀みない土地の流れを生み出した。一方のキブラーは、《[剃刀境の茂み](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%89%83%E5%88%80%E5%A2%83%E3%81%AE%E8%8C%82%E3%81%BF)》と《[魂の洞窟](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AD%82%E3%81%AE%E6%B4%9E%E7%AA%9F)》のみで止まってしまっている。


　アントニーオは《[刃の接合者](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%88%83%E3%81%AE%E6%8E%A5%E5%90%88%E8%80%85)》を出し、キブラーはトークンを《[四肢切断](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%9B%9B%E8%82%A2%E5%88%87%E6%96%AD)》するものの、続く《[幻影の像](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B9%BB%E5%BD%B1%E3%81%AE%E5%83%8F)》から生み出されるクリーチャーをどうすることもできず、かといって土地２枚で止まっているのではアントニーオに十分な攻撃をすることもできなかった。《[火柱](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%81%AB%E6%9F%B1)》がキブラーの唯一のブロッカーを除去したとき、彼は速やかにカードを片づけ、第２ゲームへと気持ちを切り替えた。


アントニーオ 1-0 キブラー


**ゲーム２**


「多少は土地が欲しいもんだ」　第２ゲームへ向けてのシャッフル中、アメリカ人はそう言って笑った。キプロスのチャンピオンはテーブルの反対側で静かなままだった。キプロスはマジックがプレイされている国の中でも小さい国だが、最大の国を相手にしてリードし、彼はその先に待つゲームへと気持ちを集中させているのだった。


　キブラーは第２ゲームでマリガンをしたが、６枚には満足していた。一方のアントニーオも、手札を戻すと次の６枚を求めていた。




|  |
| --- |
|  |
| *ダニエル・アントニーオは勝負に対し油断なく集中し続ける。* |

  
「少なくとも今回は、フェアにできそうだな」　殿堂顕彰者は笑った。双方のプレイヤーが、６枚をキープすることになった。


　第２ゲームのキブラーは、土地に問題を抱えることはなかった。第１ターンの《[アヴァシンの巡礼者](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A2%E3%83%B4%E3%82%A1%E3%82%B7%E3%83%B3%E3%81%AE%E5%B7%A1%E7%A4%BC%E8%80%85)》から、第２ターンに《[国境地帯のレインジャー](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%9B%BD%E5%A2%83%E5%9C%B0%E5%B8%AF%E3%81%AE%E3%83%AC%E3%82%A4%E3%83%B3%E3%82%B8%E3%83%A3%E3%83%BC)》で《[山](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B1%B1)》を持ってくる。《[剃刀境の茂み](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%89%83%E5%88%80%E5%A2%83%E3%81%AE%E8%8C%82%E3%81%BF)》、《[魂の洞窟](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AD%82%E3%81%AE%E6%B4%9E%E7%AA%9F)》、そして新鮮素材の土地で、彼は呪文のほとんどを唱える準備ができた。


　第２ゲームのアントニーオは静かで、《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》がキブラーのプランを首尾よく進めるのを傍観するしかできなかった。このゲームの彼の最初のプレイである自身の《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》も、ブロックできるようになる前に《[焼却](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%84%BC%E5%8D%B4)》されてしまい、ライフはひと桁に落ちた。


「奇跡」しない《[忌むべき者のかがり火](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BF%8C%E3%82%80%E3%81%B9%E3%81%8D%E8%80%85%E3%81%AE%E3%81%8B%E3%81%8C%E3%82%8A%E7%81%AB)》X=2はキブラーの軍勢を焼くのに十分ではあったが、それは大した問題ではなかった。《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》は変わらず空から攻撃し、先ほどとよく似た《[忌むべき者のかがり火](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BF%8C%E3%82%80%E3%81%B9%E3%81%8D%E8%80%85%E3%81%AE%E3%81%8B%E3%81%8C%E3%82%8A%E7%81%AB)》X=2がすぐにゲームの行方を決め、マッチは最終ゲームにもつれ込むこととなった。


アントニーオ 1-1 キブラー


**ゲーム３**


　第３ゲーム、キブラーの開幕《[スレイベンの守護者、サリア](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B9%E3%83%AC%E3%82%A4%E3%83%99%E3%83%B3%E3%81%AE%E5%AE%88%E8%AD%B7%E8%80%85%E3%80%81%E3%82%B5%E3%83%AA%E3%82%A2)》はレッドゾーンにたどり着く前に《[火柱](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%81%AB%E6%9F%B1)》に倒れ、《[刃の接合者](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%88%83%E3%81%AE%E6%8E%A5%E5%90%88%E8%80%85)》も同じ運命をたどった。しかしながら、相棒のゴーレム・トークンが残された。今回は互いのプレイヤーがマリガンをしなかったことで、勝敗を決める素晴らしいゲームが我々の前で展開されゆくようだ。


　キブラーは《[魂の洞窟](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AD%82%E3%81%AE%E6%B4%9E%E7%AA%9F)》をプレイして、「天使」を指定した。これはアントニーオにとって、潜在的だが確かな問題となる。しかし若きキプロス王者は落ち着いているようだった。《[雷口のヘルカイト](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%9B%B7%E5%8F%A3%E3%81%AE%E3%83%98%E3%83%AB%E3%82%AB%E3%82%A4%E3%83%88)》をキャストすると、レッドゾーンへ突入させる。ターン終了時、キブラーは最初の《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》をキャストした。


　アンタップすると、キブラーの手札から《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》が飛び出し、即座にそれを使って《[修復の天使](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E4%BF%AE%E5%BE%A9%E3%81%AE%E5%A4%A9%E4%BD%BF)》を《[スラーグ牙](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B9%E3%83%A9%E3%83%BC%E3%82%B0%E7%89%99)》へとアップグレードさせた。今や、キブラーは思考の間ずっと、稲妻のような速度で手札を繰っている。対照的に石像のようなアントニーオは、《[ギデオン・ジュラ](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AE%E3%83%87%E3%82%AA%E3%83%B3%E3%83%BB%E3%82%B8%E3%83%A5%E3%83%A9)》をキャストするとキブラーのゴーレム・トークンを葬った。アントニーオの《[憤怒生まれのヘルカイト](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%86%A4%E6%80%92%E7%94%9F%E3%81%BE%E3%82%8C%E3%81%AE%E3%83%98%E3%83%AB%E3%82%AB%E3%82%A4%E3%83%88)》が、このプレインズウォーカーに向かってきた《[スラーグ牙](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B9%E3%83%A9%E3%83%BC%E3%82%B0%E7%89%99)》と交換になった。これにより、キブラーには１体のビースト・トークンと《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》のみが残された。

|  |
| *ブライアン・キブラーは《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》を見事に操る。* |

  
　ブライアンは《[国境地帯のレインジャー](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%9B%BD%E5%A2%83%E5%9C%B0%E5%B8%AF%E3%81%AE%E3%83%AC%E3%82%A4%E3%83%B3%E3%82%B8%E3%83%A3%E3%83%BC)》をキャストすると、《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》を使って《[高原の狩りの達人](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%AB%98%E5%8E%9F%E3%81%AE%E7%8B%A9%E3%82%8A%E3%81%AE%E9%81%94%E4%BA%BA)》に進化させる。これに対し、アントニーオは《[瞬唱の魔道士](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%9E%AC%E5%94%B1%E3%81%AE%E9%AD%94%E9%81%93%E5%A3%AB)》から《[火柱](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%81%AB%E6%9F%B1)》を狼男に再キャストする。それぞれの交換で、キブラーは少しずつカードアドバンテージを得ていき、これにより形勢は彼の方へと、時間はかかりながらも徐々に傾いていくように見えた。《[ガヴォニーの居住区](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AC%E3%83%B4%E3%82%A9%E3%83%8B%E3%83%BC%E3%81%AE%E5%B1%85%E4%BD%8F%E5%8C%BA)》は、《[出産の殻](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%87%BA%E7%94%A3%E3%81%AE%E6%AE%BB)》なしでもキブラーのクリーチャーがすべて脅威と化すことを意味していた。実際に、アメリカ人から繰り出されるほとんどのトークンでないクリーチャーはそれぞれに「レベルアップ」していく。《[エルフの幻想家](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A8%E3%83%AB%E3%83%95%E3%81%AE%E5%B9%BB%E6%83%B3%E5%AE%B6)》は《[刃の接合者](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%88%83%E3%81%AE%E6%8E%A5%E5%90%88%E8%80%85)》になり、徐々にキブラーの戦線を埋めていく。


　アントニーオはこれに対するために、《[忌むべき者のかがり火](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BF%8C%E3%82%80%E3%81%B9%E3%81%8D%E8%80%85%E3%81%AE%E3%81%8B%E3%81%8C%E3%82%8A%E7%81%AB)》を引くしかなかった。彼のライブラリーの一番上は《[忘却の輪](https://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BF%98%E5%8D%B4%E3%81%AE%E8%BC%AA)》であったが、それは要求を満たすものではなかった。


キブラー 2-1 アントニーオ
