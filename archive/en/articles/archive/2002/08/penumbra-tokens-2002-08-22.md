Penumbra tokens
===============



 Posted in **Arcana**
 on August 22, 2002 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_authorpic_larger.jpg)
By Wizards of the Coast












We have already looked at the tokens that only "exist" in ***Magic** Online* for the [*Odyssey* block](/en/articles/archive/unseen-tokens-2002-07-30) and [*Invasion* and *Planeshift*](/en/articles/archive/hippos-and-reflections-2002-08-15), which leaves just *Apocalypse*.


All the online tokens for that set are generated when a creature goes to the graveyard. Below is a chart of the four *Apocalypse* token cards that exist only in ***Magic** Online*.




|  |  |
| --- | --- |
| Cat Token | Kavu Token |
| *Apocalypse* Cat (2/1)**Artist:** Daren Bader**Made for:**  [Penumbra Bobcat](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Penumbra+Bobcat) | *Apocalypse* Kavu (3/3)**Artist:** Arnie Swekel**Made for:**  [Penumbra Kavu](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Penumbra+Kavu) |
| Wurm Token | Angel Token |
| *Apocalypse* Wurm (6/6 trample)**Artist:** Ron Spears**Made for:**  [Penumbra Wurm](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Penumbra+Wurm) | *Apocalypse* Angel (3/3 flying)**Artist:** Glen Angus**Made for:**  [Haunted Angel](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=Haunted+Angel) |
