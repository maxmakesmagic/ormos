第８回戦：猛烈な勢いで中村 修平(日本) vs. 八十岡 翔太(日本)
===================================



 Posted in **Event Coverage**
 on August 31, 2012 






![](https://media.magic.wizards.com/styles/auth_small/public/images/person/wizards_author.jpg)
By Wizards of the Coast











　このラウンドでは昨日のモダン・ラウンドの再戦に注目しよう。そのときは、スポットライトとカメラのもと、八十岡翔太が中村修平を降している。歴代最高の構築プレイヤーとしても名前が挙がる八十岡だが、弱点のひとつはリミテッドだと言われてきた。しかしながら、マジック2013リミテッドで行われたイベント、グランプリ・ボストン＝ウースターでトップ８の実績を残し、彼の評判を曇らせるものを払拭しつつある。




|  |
| --- |
|  |
| *中村修平と八十岡翔太が、このプレイヤー選手権で二度目の激突に向かう。* |

　最近の素晴らしい実績はあるが、この日本の強豪同士の二度目の対戦では、中村に主導権があるように思える。


### ゲーム１


　八十岡はダイスロールに勝ち、先攻を選んだ。第１ターンに《[ゴブリンの付け火屋](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B4%E3%83%96%E3%83%AA%E3%83%B3%E3%81%AE%E4%BB%98%E3%81%91%E7%81%AB%E5%B1%8B)》で口火を切り、第３ターンまでにナヤ・カラーの土地を揃えたが、クリーチャーは追加できなかった。中村は《[巻物泥棒](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B7%BB%E7%89%A9%E6%B3%A5%E6%A3%92)》を戦線に加えるが、八十岡は《[平和な心](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B9%B3%E5%92%8C%E3%81%AA%E5%BF%83)》でこれを無効化した。当然のことながら、続くターンに八十岡が《[テューンの戦僧](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%86%E3%83%A5%E3%83%BC%E3%83%B3%E3%81%AE%E6%88%A6%E5%83%A7)》を加えたとき、このエンチャントを吹き飛ばすことは拒否した。


　正直なところ、私は八十岡と中村がマジックをプレイしているところを見るのが好きなのと同じくらい、彼らの観戦記事を取るのは嫌いだ。彼らのプレイは目にも留まらぬくらい速く、ついていくのは不可能に近い。ゲームは１分たりとも動きを止めることはなく、彼らはすでに中盤戦に入っていた。どちらのプレイヤーも１ターンあたり10秒をかけることがない。プレイのペースについていくために殴り書きのメモをしていたが、ついに私に追いつくための小休止が与えられた。中村は《[庇護のグリフィン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%BA%87%E8%AD%B7%E3%81%AE%E3%82%B0%E3%83%AA%E3%83%95%E3%82%A3%E3%83%B3)》と《[フェアリーの侵略者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A7%E3%82%A2%E3%83%AA%E3%83%BC%E3%81%AE%E4%BE%B5%E7%95%A5%E8%80%85)》で戦線を構築し、八十岡の試みた《[ガラクの群れ率い](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AC%E3%83%A9%E3%82%AF%E3%81%AE%E7%BE%A4%E3%82%8C%E7%8E%87%E3%81%84)》を《[本質の散乱](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%9C%AC%E8%B3%AA%E3%81%AE%E6%95%A3%E4%B9%B1)》で止めた。八十岡は《[捕食者の暴力](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%8D%95%E9%A3%9F%E8%80%85%E3%81%AE%E6%9A%B4%E5%8A%9B)》を使い、なんとか少しのダメージを通すともにアンタップ状態の《[フェアリーの侵略者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A7%E3%82%A2%E3%83%AA%E3%83%BC%E3%81%AE%E4%BE%B5%E7%95%A5%E8%80%85)》を倒した。中村は《[エイヴンの従者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A8%E3%82%A4%E3%83%B4%E3%83%B3%E3%81%AE%E5%BE%93%E8%80%85)》と《[真珠三叉矛の達人](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%9C%9F%E7%8F%A0%E4%B8%89%E5%8F%89%E7%9F%9B%E3%81%AE%E9%81%94%E4%BA%BA)》で戦線を再構築したが、それもまた長続きしなかった。




|  |
| --- |
|  |
| *中村は心配そうに対戦相手の次のアクションを待ち受けた。推定待ち時間：２秒。* |

　八十岡は攻撃し、中村は《[ゴブリンの付け火屋](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%B4%E3%83%96%E3%83%AA%E3%83%B3%E3%81%AE%E4%BB%98%E3%81%91%E7%81%AB%E5%B1%8B)》を《[真珠三叉矛の達人](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%9C%9F%E7%8F%A0%E4%B8%89%E5%8F%89%E7%9F%9B%E3%81%AE%E9%81%94%E4%BA%BA)》でブロックしてこのゴブリンを倒したが、誘発型能力でこれを失った。戦闘後、八十岡は《[捕食](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%8D%95%E9%A3%9F)》で《[エイヴンの従者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%A8%E3%82%A4%E3%83%B4%E3%83%B3%E3%81%AE%E5%BE%93%E8%80%85)》を除去した。八十岡はライフとクリーチャーの両面で押されていたが、健闘して生き残っていた。しかし、この慌ただしさが過ぎると、八十岡はガス欠に陥った。続く数ターンにわたって何も戦線に加えることができず、一方の中村は《[狩漁者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%8B%A9%E6%BC%81%E8%80%85)》と《[フェアリーの侵略者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A7%E3%82%A2%E3%83%AA%E3%83%BC%E3%81%AE%E4%BE%B5%E7%95%A5%E8%80%85)》を追加し、独壇場となった。


　最後となったターン、八十岡はカードを引くと手のひらを上に向けてテーブルを叩き、不満たっぷりの苦笑を浮かべながら手札いっぱいの土地をさらした。


中村 1-0 八十岡


### ゲーム２


　中村は第２ゲームの初手をマリガンしなければならなかったのだが、第４ターンになり、彼はそれを後悔していた。八十岡は素早いスタートを切ることなく、第３ターンの《[クウィリーオンのドライアド](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AF%E3%82%A6%E3%82%A3%E3%83%AA%E3%83%BC%E3%82%AA%E3%83%B3%E3%81%AE%E3%83%89%E3%83%A9%E3%82%A4%E3%82%A2%E3%83%89)》と続くターンの《[尊き象](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B0%8A%E3%81%8D%E8%B1%A1)》で入ったのだ。一方の中村は１枚の《[島](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B3%B6)》以上の土地を置くことができず、２個目のパーマネントを置けぬままディスカードを強いられた。プレイヤーたちは第３ゲームの準備を始めたが、このラウンドが始まってまだ10分も経っていなかった…


中村 1-1 八十岡


### ゲーム３


　この最終ゲームでは中村がまず攻勢をとった。《[狩漁者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%8B%A9%E6%BC%81%E8%80%85)》と《[極北のエイヴン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E5%8C%97%E3%81%AE%E3%82%A8%E3%82%A4%E3%83%B4%E3%83%B3)》から入ったが、エイヴンを強化するための《[平地](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B9%B3%E5%9C%B0)》がなかった。八十岡は中村が与えてくるダメージを《[群れの癒し手](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E7%BE%A4%E3%82%8C%E3%81%AE%E7%99%92%E3%81%97%E6%89%8B)》で相殺するという素晴らしい手段を見つけたが、ブロックには行けなかった。中村が攻撃を試みたところで、八十岡は《[チャンドラの憤怒](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%81%E3%83%A3%E3%83%B3%E3%83%89%E3%83%A9%E3%81%AE%E6%86%A4%E6%80%92)》で《[極北のエイヴン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E5%8C%97%E3%81%AE%E3%82%A8%E3%82%A4%E3%83%B4%E3%83%B3)》を除去しようとするが、中村は《[送還](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E9%80%81%E9%82%84)》でこれを救った。


　八十岡は《[松明の悪鬼](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%9D%BE%E6%98%8E%E3%81%AE%E6%82%AA%E9%AC%BC)》、《[ガラクの群れ率い](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AC%E3%83%A9%E3%82%AF%E3%81%AE%E7%BE%A4%E3%82%8C%E7%8E%87%E3%81%84)》と戦線にさらなるクリーチャーを追加し始めた。中村は《[極北のエイヴン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%A5%B5%E5%8C%97%E3%81%AE%E3%82%A8%E3%82%A4%E3%83%B4%E3%83%B3)》を再プレイするが、八十岡は《[捕食](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%8D%95%E9%A3%9F)》でこれを撃ち落とした。そしてゲームは対照的な方向に動いていく。八十岡のライフは上昇、中村のライフは下降し、八十岡の戦線は埋め尽くされ中村のそれは空になっていった。中村の力ない《[オドリックの十字軍](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%82%AA%E3%83%89%E3%83%AA%E3%83%83%E3%82%AF%E3%81%AE%E5%8D%81%E5%AD%97%E8%BB%8D)》を八十岡が《[溶岩噴火](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E6%BA%B6%E5%B2%A9%E5%99%B4%E7%81%AB)》で除去したのは終焉の前兆か、攻撃してライフを８に落とす。




|  |
| --- |
|  |
| *たくさんのカードを散らばらせてしまう八十岡* |

　中村は《[フェアリーの侵略者](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%83%95%E3%82%A7%E3%82%A2%E3%83%AA%E3%83%BC%E3%81%AE%E4%BE%B5%E7%95%A5%E8%80%85)》を瞬速で出してブロックし、八十岡の攻勢を遅らせようとするのだが、この攻撃でライフは２。《[安全な道](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%AE%89%E5%85%A8%E3%81%AA%E9%81%93)》でもう１ターンを稼ぐが、そのターンに《[かき回すゴブリン](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E3%81%8B%E3%81%8D%E5%9B%9E%E3%81%99%E3%82%B4%E3%83%96%E3%83%AA%E3%83%B3)》が《[帆凧](http://gatherer.wizards.com/Pages/Card/Details.aspx?name=%E5%B8%86%E5%87%A7)》を見つけ、それは勝負を決めるに十分だった。


中村 1-2 八十岡


  
(Tr. Yusuke Yoshikawa)
