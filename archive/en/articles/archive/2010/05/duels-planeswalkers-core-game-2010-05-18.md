Features:


* Eight playable decks with seventeen unlockable cards each
* Eight puzzling challenge levels

### Achievements




|  |  |  |  |
| --- | --- | --- | --- |
|  | **Name** | **Description** | **Points** |
|  | The Spark | Win a game | 20 |
|  | Life Preserver | Increase your life total to 30 or more in single player, or 60 or more in Two-Headed Giant | 10 |
|  | Magic: The Puzzling | Complete a challenge | 10 |
|  | Devastator | Deal 20 combat damage to a player in a single turn in a game | 10 |
|  | Defender of Foriys | Win a Two-Headed Giant game | 20 |
|  | Spellslinger | Defeat someone who has earned the Spellslinger achievement | 0 |
|  | Duelist | Win 5 ranked Xbox LIVE matches | 10 |
|  | Epic Conclusion | Defeat Tezzeret in the final single-player campaign battle | 40 |
|  | Puzzling Master | Complete all challenges | 20 |
|  | Master of Artifice | Defeat Tezzeret & Tezzeret in the cooperative campaign | 10 |
|  | Duels Master | Win 10 Xbox LIVE matches | 20 |
|  | Planeswalker | Unlock 100% | 30 |
